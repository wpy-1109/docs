OSS Vector Search enables you to store, index, and search vector embeddings directly in OSS buckets. This supports AI applications like semantic search, recommendation systems, and retrieval-augmented generation (RAG).

## Key features

| Feature | Description |
|---------|-------------|
| **Vector storage** | Store high-dimensional vectors alongside object metadata |
| **Similarity search** | Find nearest neighbors using cosine, L2, or inner product distance |
| **Scalable** | Handles billions of vectors with distributed indexing |
| **Integrated** | Works with existing OSS buckets and access control |

## Use cases

- **Semantic search**: Search documents, images, or products by meaning
- **RAG applications**: Retrieve relevant context for LLM prompts
- **Recommendation engines**: Find similar items based on embeddings
- **Image similarity**: Visual search across image collections
- **Anomaly detection**: Identify outliers in embedding space

## Getting started

1. Enable vector indexing on your bucket
2. Upload objects with vector embeddings
3. Create a vector index
4. Run similarity queries

See [Quickstart](/tools/vectors/quickstart) for a hands-on tutorial.
