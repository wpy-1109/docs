This quickstart walks you through storing and searching vector embeddings in OSS.

## Prerequisites

- An OSS bucket with vector search enabled
- ossutil or an OSS SDK installed
- Vector embeddings from your ML model

## Step 1: enable vector search

Enable vector search for your bucket through the OSS console or API.

## Step 2: upload objects with embeddings

When uploading objects, include vector embeddings as metadata:

```python
import oss2

auth = oss2.Auth('your-access-key-id', 'your-access-key-secret')
bucket = oss2.Bucket(auth, 'https://oss-cn-hangzhou.aliyuncs.com', 'your-bucket')

# Upload with vector metadata
headers = {
    'x-oss-meta-vector': '[0.1, 0.2, 0.3, ...]'  # Your embedding vector
}
bucket.put_object('documents/doc1.txt', 'Document content', headers=headers)
```

## Step 3: create a vector index

Create an index specifying the vector dimension and distance metric:

```bash
ossutil api put-bucket-vector-index \
  --bucket your-bucket \
  --dimension 768 \
  --metric cosine
```

## Step 4: search

Query for similar vectors:

```python
# Search for similar documents
results = bucket.vector_search(
    query_vector=[0.1, 0.2, 0.3, ...],
    top_k=10,
    metric='cosine'
)

for result in results:
    print(f"Key: {result.key}, Score: {result.score}")
```
