---
title: 'ossutil Overview'
description: 'Command-line tool for managing Alibaba Cloud OSS buckets and objects'
---

ossutil is a command-line tool that simplifies the management of OSS across different operating systems. It supports fast uploads, downloads, synchronization, and bucket management.

## Installation

<Tabs>
  <Tab title="Linux">
    <Steps>
      <Step title="Install unzip">
        ```bash
        # Alibaba Cloud Linux / CentOS
        sudo yum install -y unzip

        sudo apt install -y unzip
        ```
      </Step>
      <Step title="Download ossutil">
        ```bash
        curl -o ossutil-2.2.0-linux-amd64.zip \
          https://gosspublic.alicdn.com/ossutil/v2/2.2.0/ossutil-2.2.0-linux-amd64.zip
        ```

      </Step>
      <Step title="Extract and install">
        ```bash
        unzip ossutil-2.2.0-linux-amd64.zip
        cd ossutil-2.2.0-linux-amd64
        chmod 755 ossutil
        sudo mv ossutil /usr/local/bin/
        ```
      </Step>
      <Step title="Verify installation">
        ```bash
        ossutil version
        ```
      </Step>
    </Steps>

## Configuration

### Interactive setup

```bash
ossutil config
```

Follow the prompts to enter:
1. Configuration file path (default: `~/.ossutilconfig`)
2. AccessKey ID
3. AccessKey Secret
4. Region ID (e.g., `cn-hangzhou`)
5. Custom endpoint (optional)

### Configuration file

The configuration file uses INI format:

```ini
[default]
accessKeyID = "LTAI****"
accessKeySecret = "****"
region = cn-hangzhou
```

### Configuration precedence

Command-line flags > Environment variables > Configuration file

### Environment variables

| Variable | Description |
|----------|-------------|
| `OSS_ACCESS_KEY_ID` | AccessKey ID |
| `OSS_ACCESS_KEY_SECRET` | AccessKey Secret |
| `OSS_SESSION_TOKEN` | STS session token |
| `OSS_REGION` | Default region |
| `OSS_ENDPOINT` | Custom endpoint |

## Quick start

```bash
# Create a bucket
ossutil mb oss://my-bucket

# Upload a file
ossutil cp localfile.txt oss://my-bucket/

# List objects
ossutil ls oss://my-bucket

# Download a file
ossutil cp oss://my-bucket/localfile.txt ./downloaded.txt

# Delete a file
ossutil rm oss://my-bucket/localfile.txt

# Delete a bucket
ossutil rb oss://my-bucket
```

See [Command Reference](/tools/ossutil/commands) for the complete list of commands and options.
