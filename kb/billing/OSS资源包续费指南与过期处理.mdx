# OSS资源包续费指南与过期处理

OSS 资源包到期前需要及时续费，否则费用将切换为按量付费。已过期的资源包无法直接续费，需要重新购买。

## 适用场景

- 收到资源包即将到期的通知
- 资源包已过期，想继续使用
- 想了解续费的具体操作

## 操作步骤

### 续费未过期的资源包

1. 登录 [资源包管理页面](https://usercenter2.aliyun.com/resource-pack/prepaid)
2. 找到要续费的 OSS 资源包
3. 点击"续费"按钮
4. 选择续费时长，确认支付

> **注意**：续费只能选择 1 年期限。如果想购买更长期限（如 3 年、5 年），需要新购资源包。

### 过期资源包的处理

已过期的资源包无法续费，需要：

1. 前往 [OSS 资源包购买页面](https://common-buy.aliyun.com/?commodityCode=ossbag)
2. 选择与原资源包相同的类型和规格
3. 重新购买即可

> 重新购买不影响已有数据。新旧资源包之间的空窗期，费用按量从账户余额扣除。

## 资源包到期后的影响

| 影响 | 说明 |
|------|------|
| 数据是否丢失 | 不会，数据不受影响 |
| 服务是否中断 | 不会，自动切换为按量付费 |
| 费用变化 | 从资源包抵扣变为按量扣余额 |
| 欠费风险 | 如果余额不足，可能导致欠费停服 |

## 续费建议

- 开启到期自动续费，避免忘记续费
- 比较续费价格和新购价格，有时新购更划算
- 如果用量有变化，趁换购的机会调整规格

## 常见问题

**Q：续费后已有数据会丢失吗？**
不会。资源包只影响计费方式，不影响数据。

**Q：买了新资源包，但续费按钮是灰色的？**
资源包过期后续费按钮不可用，这是正常的。直接新购即可。

**Q：可以一次买多年吗？**
可以。购买页面支持选择不同时长。注意续费通常只支持 1 年，但新购可以选更长期限。

**Q：续费 5 年和新购 5 年价格一样吗？**
通常新购长期限有更多折扣。建议对比两种方式的价格。
