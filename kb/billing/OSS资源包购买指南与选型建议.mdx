# OSS资源包购买指南与选型建议

购买 OSS 资源包可以预付费抵扣后付费账单，降低使用成本。但 OSS 有多种计费项，需要选购正确类型的资源包才能生效。

## 适用场景

- 首次使用 OSS，需要了解应该购买哪些资源包
- 已有 OSS 使用量，想通过资源包降低成本
- 收到"计费项用量超出资源包规格"的提醒，不确定如何处理

## OSS 主要计费项与对应资源包

OSS 的费用由多个计费项组成，每种计费项需要购买对应的资源包：

| 计费项 | 对应资源包 | 说明 |
|--------|-----------|------|
| 存储容量（标准-本地冗余） | 标准存储包（本地冗余） | 最常见的存储类型 |
| 存储容量（标准-同城冗余） | 标准存储包（同城冗余） | 更高可用性，价格更高 |
| 存储容量（低频/归档/冷归档） | 对应类型的存储包 | 按实际使用的存储类型购买 |
| 外网流出流量 | 下行流量包 | 用户通过公网下载/访问 OSS 文件时产生 |
| CDN 回源流出流量 | 回源流量包 | 使用 CDN 加速时，CDN 从 OSS 拉取文件产生 |
| 请求次数 | 请求次数包 | GET/PUT 等 API 调用产生 |

## 操作步骤

### 1. 确认当前使用的存储类型和地域

登录 [OSS 控制台](https://oss.console.aliyun.com/)，查看各 Bucket 的存储类型（标准/低频/归档）和冗余类型（本地冗余/同城冗余）。

### 2. 查看当前费用构成

进入 [费用中心 > 账单详情](https://billing-cost.console.aliyun.com/finance/expense-report/expense-detail-by-instance)，筛选"对象存储 OSS"，查看各计费项的费用占比。

### 3. 购买对应资源包

前往 [OSS 资源包购买页面](https://common-buy.aliyun.com/?commodityCode=ossbag)，根据实际用量选择：

- **存储包**：选择与 Bucket 一致的存储类型和冗余类型
- **流量包**：根据外网流出流量大小选择
- **地域**：选择"中国大陆"通用包，或选择具体地域的包

### 4. 确认资源包生效

资源包购买后约 1 小时生效。资源包只能抵扣购买后产生的费用，不能追溯抵扣之前的账单。

## 常见选购错误

**存储类型不匹配**：Bucket 使用的是"标准-本地冗余"，但购买了"标准-同城冗余"的存储包，无法抵扣。请在控制台确认 Bucket 的实际冗余类型。

**只买存储包不买流量包**：存储包只抵扣存储费用。用户通过公网下载文件会产生外网流出流量费用（闲时 0.25 元/GB，忙时 0.50 元/GB），需要单独购买下行流量包。

**资源包容量不足**：当实际用量超出资源包规格时，超出部分按量计费。可以在 [资源包管理页面](https://usercenter2.aliyun.com/resource-pack/prepaid) 查看使用情况。

## 资源包叠加规则

- 标准-同城冗余存储包：支持叠加购买
- 标准-本地冗余存储包：同类型不支持叠加，但"中国大陆"通用包可与单地域包叠加
- 不同类型的资源包互不影响，可同时购买存储包和流量包

## 资源包到期后怎么办

资源包到期后自动失效，不影响 OSS 服务本身。费用会切换为按量付费，从账户余额中扣除。建议在到期前续购或购买新的资源包。

> **注意**：过期的资源包无法直接续费，需要重新购买新的资源包。
