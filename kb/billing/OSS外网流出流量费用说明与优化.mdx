# OSS外网流出流量费用说明与优化

通过公网访问或下载 OSS 中的文件会产生外网流出流量费用，这是除存储费用外最常见的 OSS 计费项。

## 常见症状

- 买了存储资源包但仍在扣费，扣费来自"外网流出流量"
- 流量费用高于预期
- 不清楚流量费用是怎么产生的

## 什么操作会产生外网流出流量

| 操作 | 是否产生外网流出流量 |
|------|-------------------|
| 通过公网下载 OSS 文件 | 是 |
| 通过浏览器访问 OSS 中的图片/视频 | 是 |
| 同地域 ECS 通过内网 Endpoint 访问 | 否（免费） |
| 上传文件到 OSS | 否（上传不收流量费） |
| CDN 回源拉取 OSS 文件 | 不算外网流出，算 CDN 回源流量 |

## 流量收费标准

OSS 外网流量（下行）费用区分闲时和忙时：

- **闲时**（00:00-08:00）：0.25 元/GB
- **忙时**（08:00-24:00）：0.50 元/GB

> 上行流量（上传）和内网流量免费。

## 解决方案

### 方案一：购买下行流量包

购买 [OSS 下行流量包](https://common-buy.aliyun.com/?commodityCode=ossbag) 可以抵扣外网流出流量费用。

选购建议：
- 在 [账单详情](https://billing-cost.console.aliyun.com/) 查看每月外网流出流量用量
- 选择与实际用量匹配的规格
- 流量包按月重置，每月都有额定流量

### 方案二：使用 CDN 加速

如果文件被大量公网用户访问，建议使用 CDN 加速：

- CDN 回源流量的价格低于 OSS 外网流出流量
- CDN 还可以提升访问速度
- 配置 CDN 后，流量费用从"OSS 外网流出流量"变为"CDN 流量"+"OSS 回源流量"

### 方案三：使用内网访问

如果是 ECS 或其他阿里云产品访问 OSS，使用内网 Endpoint 可以完全免除流量费用：

- 同地域内网 Endpoint 格式：`oss-cn-<region>-internal.aliyuncs.com`
- 例如：`oss-cn-hangzhou-internal.aliyuncs.com`

> **注意**：必须是同地域、同账号下的内网才免费。跨地域走内网仍会产生费用。

## 如何查看流量使用明细

1. 登录 OSS 控制台，选择目标 Bucket
2. 在"数据概览"或"基础数据"中查看流量统计
3. 如需更详细的访问日志，开启 [实时日志查询](https://help.aliyun.com/zh/oss/user-guide/real-time-log-query/) 功能

## 预防措施

- 业务系统中使用内网 Endpoint 访问 OSS
- 对公网大流量场景配合 CDN 使用
- 定期检查流量用量，发现异常及时排查
