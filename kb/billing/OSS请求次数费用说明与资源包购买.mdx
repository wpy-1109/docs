# OSS请求次数费用说明与资源包购买

OSS 的每次 API 调用（上传、下载、列举文件等）都会产生请求次数费用。了解请求计费规则可以避免意外扣费。

## 什么是请求次数费用

每次对 OSS 的 API 操作都算一次请求，包括：

| 操作 | 请求类型 | 说明 |
|------|---------|------|
| 上传文件 | PUT | 每上传一个文件算一次 |
| 下载文件 | GET | 每下载一个文件算一次 |
| 列举文件 | GET | 每次 ListObjects 算一次（每次最多返回 1000 个文件） |
| 删除文件 | DELETE | 每次删除算一次 |
| 获取文件信息 | HEAD | 每次查询算一次 |
| 目录统计 | GET | 统计 1 万个文件需要约 10-100 次请求 |

## 费用标准

- **标准存储类型**：0.01 元/万次
- **低频/归档存储类型**：非标准请求价格更高
- **免费额度**：每月每地域标准类型前 500 万次免费（仅限标准存储）

> 通过默认域名上传文件不产生流量费用，但会产生请求次数费用。

## 常见问题

**Q：上传文件到 OSS 会产生费用吗？**
会产生 PUT 请求费用（0.01 元/万次）。上传流量免费，但请求次数收费。

**Q：请求次数包怎么买？**
在 [OSS 资源包购买页面](https://common-buy.aliyun.com/?commodityCode=ossbag) 选择"请求次数包"，选择标准或非标准类型。

**Q：访问图片/视频也算请求吗？**
是的。每次通过 URL 访问 OSS 中的文件（图片、视频、文档等）都算一次 GET 请求。

**Q：设置生命周期规则会产生请求费用吗？**
生命周期规则执行转换或删除操作时会产生请求费用，但通常费用很低。
