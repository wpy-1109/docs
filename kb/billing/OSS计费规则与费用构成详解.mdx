# OSS计费规则与费用构成详解

OSS 采用按量付费模式，费用由存储、流量、请求次数等多个计费项组成。了解各计费项的含义可以帮助您更好地控制成本。

## OSS 费用构成

OSS 的总费用 = 存储费用 + 流量费用 + 请求费用 + 数据处理费用（可选）

### 1. 存储费用

按照存储容量和存储时长计费，每小时结算一次。

| 存储类型 | 参考价格（中国大陆） | 适用场景 |
|---------|-------------------|---------|
| 标准存储（本地冗余） | 0.12 元/GB/月 | 频繁访问的热数据 |
| 标准存储（同城冗余） | 0.15 元/GB/月 | 需要更高可用性的热数据 |
| 低频访问存储 | 0.08 元/GB/月 | 每月访问 1-2 次的数据 |
| 归档存储 | 0.033 元/GB/月 | 极少访问，需长期保存 |
| 冷归档存储 | 0.015 元/GB/月 | 极少访问，合规归档 |
| 深度冷归档存储 | 0.0045 元/GB/月 | 几乎不访问，最低成本归档 |

> 实际价格以 [OSS 产品定价页](https://www.aliyun.com/price/detail/oss) 为准，不同地域价格略有差异。

### 2. 流量费用

| 流量类型 | 是否收费 | 价格参考 |
|---------|---------|---------|
| 外网流出（下行） | 收费 | 闲时 0.25 元/GB，忙时 0.50 元/GB |
| 外网流入（上传） | 免费 | - |
| 内网流量（同地域） | 免费 | - |
| CDN 回源流出 | 收费 | 0.15 元/GB |
| 跨区域复制流量 | 收费 | 按源地域计费 |

### 3. 请求费用

| 请求类型 | 价格参考 |
|---------|---------|
| PUT/POST/DELETE 等写类请求 | 0.01 元/万次 |
| GET/HEAD 等读类请求 | 0.01 元/万次 |

### 4. 数据处理费用（可选）

图片处理、视频截帧、数据索引等增值功能会额外收费。

## 计费方式

### 按量付费（后付费）

OSS 默认采用按量付费，按实际使用量每小时结算，从账户余额中扣除。

### 资源包（预付费）

购买资源包可以预付费抵扣对应的计费项，通常比按量付费更划算。

### ECS 挂载 OSS 的计费

通过 ossfs 或 CloudDFS 将 OSS 挂载到 ECS：
- **同地域内网访问**：免流量费，仅收取存储费和请求费
- **跨地域访问**：收取流量费
- 挂载操作本身不额外收费

## 常见问题

**Q：开通 OSS 会产生费用吗？**
开通 OSS 服务本身免费。只有在创建 Bucket 并存储数据后才开始计费。

**Q：创建空 Bucket 会收费吗？**
空 Bucket 不收取存储费用。但如果有 API 请求（如 ListObjects），会产生少量请求费用。

**Q：OSS 有免费额度吗？**
新开通 OSS 服务后有限时免费试用额度，包括一定的存储容量和请求次数。免费额度用完后按量计费。

**Q：每天费用怎么查？**
登录 [费用中心 > 账单详情](https://billing-cost.console.aliyun.com/)，选择"对象存储 OSS"，可查看每日费用明细。
