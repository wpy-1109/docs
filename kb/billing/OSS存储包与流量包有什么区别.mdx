# OSS存储包与流量包有什么区别

存储包抵扣存储费用，流量包抵扣流量费用。两者互不替代，需要根据实际使用情况分别购买。

## 详细说明

### 存储包

- **抵扣范围**：OSS 存储容量费用
- **计算方式**：总量恒定型，每小时可抵扣固定容量
- **场景**：Bucket 中存放了数据，产生的存储费用
- **注意**：需要选择与 Bucket 一致的存储类型（标准/低频/归档）和冗余类型（本地冗余/同城冗余）

### 流量包（下行流量包）

- **抵扣范围**：外网流出流量费用
- **计算方式**：每月有固定流量额度，用完按量计费
- **场景**：用户通过公网下载或访问 OSS 文件时产生的流量
- **注意**：上传流量免费，只有下行流量需要购买流量包

### 其他类型资源包

| 资源包 | 抵扣计费项 |
|--------|----------|
| 请求次数包 | GET/PUT 等 API 请求费用 |
| 回源流量包 | CDN 从 OSS 拉取文件的回源流量费用 |
| 传输加速包 | 传输加速功能的流量费用 |

## 如何判断需要买哪种

1. **只存数据、不通过公网下载**：只买存储包即可
2. **既存数据又通过公网下载**：存储包 + 下行流量包
3. **使用了 CDN 加速**：存储包 + 回源流量包
4. **大量 API 调用**：再加一个请求次数包

## 常见误解

**"买了存储包就什么都包了"** -- 错误。存储包只管存储费用，流量和请求需要另外买。

**"买了流量包就不用买存储包了"** -- 错误。流量包只管流量费用，存储费用仍需存储包或按量付费。

**"只要买一个资源包就够了"** -- 看情况。如果只产生存储费用（没有外网访问），一个存储包就够。如果有外网访问，至少需要存储包 + 流量包。
