---
title: 'ListObjectVersions'
description: 'List all versions of objects in a bucket'
---

Lists all versions of objects in a versioning-enabled bucket. Also known as `GetBucketVersions`.

## Request

```
GET /?versions HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `prefix` | String | No | Filter by key prefix. |
| `delimiter` | String | No | Character to group keys. |
| `key-marker` | String | No | Start listing from this key. |
| `version-id-marker` | String | No | Start listing from this version ID. |
| `max-keys` | Integer | No | Maximum results. Default: 100. Max: 1000. |

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ListVersionsResult>
  <Name>my-bucket</Name>
  <IsTruncated>false</IsTruncated>
  <Version>
    <Key>myobject.txt</Key>
    <VersionId>CAEQNhiBgMDJ****</VersionId>
    <IsLatest>true</IsLatest>
    <LastModified>2025-04-17T03:15:40.000Z</LastModified>
    <ETag>"D41D8CD98F00B204****"</ETag>
    <Size>344606</Size>
    <StorageClass>Standard</StorageClass>
  </Version>
  <DeleteMarker>
    <Key>deleted-object.txt</Key>
    <VersionId>CAEQNhiBgIC4****</VersionId>
    <IsLatest>true</IsLatest>
    <LastModified>2025-04-17T04:00:00.000Z</LastModified>
  </DeleteMarker>
</ListVersionsResult>
```
