---
title: 'PutBucketVersioning'
description: 'Enable or suspend versioning for a bucket'
---

Sets the versioning state of a bucket. Once enabled, versioning cannot be disabled -- it can only be suspended.

## Request

```
PUT /?versioning HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<VersioningConfiguration>
  <Status>Enabled</Status>
</VersioningConfiguration>
```

| Element | Type | Description |
|---------|------|-------------|
| `Status` | String | `Enabled` to enable versioning, `Suspended` to suspend it. |

## Response

**Success:** HTTP 200 with empty body.

<Note>
When versioning is enabled, overwriting or deleting an object creates a new version instead of replacing the original. Suspended versioning stops creating new versions but preserves existing ones.
</Note>
