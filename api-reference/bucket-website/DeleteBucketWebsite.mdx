---
title: 'DeleteBucketWebsite'
description: 'Disable static website hosting for a bucket'
---

Disables static website hosting for a bucket and removes all routing rules.

## Request

```
DELETE /?website HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.
