---
title: 'GetBucketArchiveDirectRead'
description: 'Query whether archive direct read is enabled'
---


## Request

```
GET /?bucketArchiveDirectRead HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ArchiveDirectReadConfiguration>
  <Enabled>true</Enabled>
</ArchiveDirectReadConfiguration>
```
