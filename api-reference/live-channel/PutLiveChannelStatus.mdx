---
title: 'PutLiveChannelStatus'
description: 'Enable or disable a LiveChannel'
---


## Request

```
PUT /{ChannelName}?live&status={Status} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 200.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchLiveChannel` | 404 | The LiveChannel does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
