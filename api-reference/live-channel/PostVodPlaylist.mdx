title: 'PostVodPlaylist'
description: 'Generate a VOD playlist for a time range'
---

Generate a VOD playlist for a time range.

## Request

```
POST /{ChannelName}?vod HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 200.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchLiveChannel` | 404 | The LiveChannel does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
