---
title: 'ListLiveChannel'
description: 'List all LiveChannels in a bucket'
---

List all LiveChannels in a bucket.

## Request

```
GET /{ChannelName}?live HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 200.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchLiveChannel` | 404 | The LiveChannel does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
