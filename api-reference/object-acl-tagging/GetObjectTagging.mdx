---
title: 'GetObjectTagging'
description: 'Query tags of an object'
---


## Request

```
GET /{ObjectName}?tagging HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Tagging>
  <TagSet>
    <Tag>
      <Key>environment</Key>
      <Value>production</Value>
    </Tag>
  </TagSet>
</Tagging>
```
