---
title: 'PutObjectAcl'
description: 'Set the ACL of an object'
---

Sets the access control list (ACL) of an object.

## Request

```
PUT /{ObjectName}?acl HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
x-oss-object-acl: {ACL}
```

### ACL values

| Value | Description |
|-------|-------------|
| `default` | Inherits bucket ACL. |
| `private` | Only the owner can read/write. |
| `public-read` | Anyone can read. Only the owner can write. |
| `public-read-write` | Anyone can read/write. |

## Response

**Success:** HTTP 200 with empty body.
