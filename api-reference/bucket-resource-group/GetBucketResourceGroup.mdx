---
title: 'GetBucketResourceGroup'
description: 'Query the resource group of a bucket'
---


## Request

```
GET /?resourceGroup HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<BucketResourceGroupConfiguration>
  <ResourceGroupId>rg-aek27tc****</ResourceGroupId>
</BucketResourceGroupConfiguration>
```
