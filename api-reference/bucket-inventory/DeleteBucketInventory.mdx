---
title: 'DeleteBucketInventory'
description: 'Delete an inventory task from a bucket'
---

Deletes a specified inventory task from a bucket.

## Request

```
DELETE /?inventory&inventoryId={InventoryId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.
