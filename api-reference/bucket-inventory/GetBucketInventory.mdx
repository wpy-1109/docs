---
title: 'GetBucketInventory'
description: 'Query a specific inventory task'
---

Queries a specified inventory task in a bucket.

## Request

```
GET /?inventory&inventoryId={InventoryId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

Returns the inventory configuration XML for the specified task.
