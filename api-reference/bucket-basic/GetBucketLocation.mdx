---
title: 'GetBucketLocation'
description: 'Query the region of a bucket'
---

Returns the region (Location) of the specified bucket.

## Request

```
GET /?location HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<LocationConstraint>oss-cn-hangzhou</LocationConstraint>
```

## Example

```http
GET /?location HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Date: Thu, 17 Apr 2025 03:15:40 GMT
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

```http
HTTP/1.1 200 OK
x-oss-request-id: 534B371674E88A4D8906****

<?xml version="1.0" encoding="UTF-8"?>
<LocationConstraint>oss-cn-hangzhou</LocationConstraint>
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The specified bucket does not exist. |
| `AccessDenied` | 403 | You do not have permission to access this bucket. |
