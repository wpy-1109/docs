---
title: 'GetMetaQueryStatus'
description: 'Query the metadata index status of a bucket'
---

Queries the metadata index status of a bucket.

## Request

```
GET /?metaQuery HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<MetaQueryStatus>
  <State>Running</State>
  <Phase>FullScanning</Phase>
  <CreateTime>2025-04-17T03:15:40.000Z</CreateTime>
  <UpdateTime>2025-04-17T04:00:00.000Z</UpdateTime>
</MetaQueryStatus>
```
