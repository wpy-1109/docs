---
title: 'GetBucketLifecycle'
description: 'Query lifecycle rules for a bucket'
---

Queries the lifecycle rules configured for a bucket.

## Request

```
GET /?lifecycle HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

Returns the lifecycle configuration XML. See [PutBucketLifecycle](/api-reference/bucket-lifecycle/PutBucketLifecycle) for element descriptions.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `NoSuchLifecycle` | 404 | No lifecycle rules are configured. |
