---
title: 'GetStyle'
description: 'Query a specific image style'
---


## Request

```
GET /?style&styleName={StyleName} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Style>
  <Name>thumbnail</Name>
  <Content>image/resize,w_200,h_200/quality,q_80</Content>
  <CreateTime>2025-04-17T03:15:40.000Z</CreateTime>
  <LastModifyTime>2025-04-17T03:15:40.000Z</LastModifyTime>
</Style>
```
