---
title: 'GetAccessPointPolicy'
description: 'Query the policy of an access point'
---

Queries the access policy of a bucket access point.

## Request

```
GET /?accessPointPolicy HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
x-oss-access-point-name: {AccessPointName}
```

## Response

Returns the policy as a JSON document in the response body.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchAccessPoint` | 404 | The access point does not exist. |
| `NoSuchAccessPointPolicy` | 404 | No policy is configured. |
