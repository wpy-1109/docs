---
title: 'PutAccessPointConfigForObjectProcess'
description: 'Modifies the configuration of an Object FC access point.'
---


## Request

```
PUT /?accessPointConfigForObjectProcess HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 200.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchAccessPointForObjectProcess` | 404 | The Object FC access point does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
