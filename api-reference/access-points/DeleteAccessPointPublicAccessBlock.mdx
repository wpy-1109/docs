---
title: 'DeleteAccessPointPublicAccessBlock'
description: 'Delete Block Public Access for an access point'
---

Deletes the Block Public Access configuration for a specified access point.

## Request

```
DELETE /?publicAccessBlock HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
x-oss-access-point-name: {AccessPointName}
```


## Response


**Success:** HTTP 204 with empty body.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchAccessPoint` | 404 | The access point does not exist. |
