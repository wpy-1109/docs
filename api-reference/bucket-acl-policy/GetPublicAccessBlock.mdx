---
title: 'GetPublicAccessBlock'
description: 'Query the global Block Public Access configuration'
---


## Request

```
GET /?publicAccessBlock HTTP/1.1
Host: oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<PublicAccessBlockConfiguration>
  <BlockPublicAccess>true</BlockPublicAccess>
</PublicAccessBlockConfiguration>
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchPublicAccessBlockConfiguration` | 404 | No global configuration exists. |
