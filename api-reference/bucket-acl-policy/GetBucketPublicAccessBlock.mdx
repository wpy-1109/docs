---
title: 'GetBucketPublicAccessBlock'
description: 'Query the Block Public Access configuration of a bucket'
---


## Request

```
GET /?publicAccessBlock HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<PublicAccessBlockConfiguration>
  <BlockPublicAccess>true</BlockPublicAccess>
</PublicAccessBlockConfiguration>
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `NoSuchPublicAccessBlockConfiguration` | 404 | No configuration exists for this bucket. |
