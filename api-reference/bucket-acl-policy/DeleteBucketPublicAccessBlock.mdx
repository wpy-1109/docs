---
title: 'DeleteBucketPublicAccessBlock'
description: 'Delete the Block Public Access configuration of a bucket'
---

Deletes the Block Public Access configuration of a specified bucket.

## Request

```
DELETE /?publicAccessBlock HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
