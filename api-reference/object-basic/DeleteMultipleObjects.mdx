---
title: 'DeleteMultipleObjects'
description: 'Delete multiple objects in a single request'
---

Deletes up to 1,000 objects in a single request.

## Request

```
POST /?delete HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
Content-MD5: {MD5}
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Delete>
  <Quiet>false</Quiet>
  <Object>
    <Key>object1.txt</Key>
  </Object>
  <Object>
    <Key>object2.txt</Key>
    <VersionId>CAEQNhiBgIC4****</VersionId>
  </Object>
</Delete>
```

| Element | Type | Description |
|---------|------|-------------|
| `Quiet` | Boolean | `true` returns only errors. `false` returns all results. |
| `Object.Key` | String | Object key to delete. |
| `Object.VersionId` | String | Optional version ID for versioned objects. |

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<DeleteResult>
  <Deleted>
    <Key>object1.txt</Key>
  </Deleted>
  <Deleted>
    <Key>object2.txt</Key>
    <DeleteMarker>true</DeleteMarker>
    <DeleteMarkerVersionId>CAEQNhiBgIC4****</DeleteMarkerVersionId>
  </Deleted>
</DeleteResult>
```

<Note>
The `Content-MD5` header is required for this operation to ensure request body integrity.
</Note>
