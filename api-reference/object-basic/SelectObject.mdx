---
title: 'SelectObject'
description: 'Query object content using SQL expressions'
---

Executes a SQL statement against a CSV or JSON object and returns matching results, avoiding downloading the entire object.

## Request

```
POST /{ObjectName}?x-oss-process=csv/select HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<SelectRequest>
  <Expression>c2VsZWN0ICogZnJvbSBvc3NvYmplY3Qgd2hlcmUgX3RpbWUgPiAxMDAwMDAw</Expression>
  <InputSerialization>
    <CSV>
      <FileHeaderInfo>Use</FileHeaderInfo>
      <RecordDelimiter>\n</RecordDelimiter>
      <FieldDelimiter>,</FieldDelimiter>
    </CSV>
  </InputSerialization>
  <OutputSerialization>
    <CSV>
      <RecordDelimiter>\n</RecordDelimiter>
      <FieldDelimiter>,</FieldDelimiter>
    </CSV>
  </OutputSerialization>
</SelectRequest>
```

| Element | Type | Description |
|---------|------|-------------|
| `Expression` | String | Base64-encoded SQL expression (e.g., `select * from ossobject where _time > 1000000`). |
| `InputSerialization` | Element | Input format configuration (CSV or JSON). |
| `OutputSerialization` | Element | Output format configuration. |

## Supported SQL

- `SELECT` with column references or `*`
- `WHERE` with comparison operators (`=`, `!=`, `>`, `<`, `>=`, `<=`)
- `LIKE`, `IN`, `BETWEEN`, `AND`, `OR`, `NOT`
- Aggregate functions: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`

## Response

Returns the query results in the specified output format.
