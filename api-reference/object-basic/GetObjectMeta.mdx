---
title: 'GetObjectMeta'
description: 'Get basic object metadata (ETag, size, last modified)'
---

Returns basic metadata of an object (ETag, size, last modified time) without the object content. Lighter-weight than HeadObject.

## Request

```
HEAD /{ObjectName}?objectMeta HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```http
HTTP/1.1 200 OK
ETag: "D41D8CD98F00B204E9800998ECF8****"
Content-Length: 1024
Last-Modified: Thu, 17 Apr 2025 03:15:40 GMT
x-oss-version-id: CAEQNhiBgMDJ****
```
