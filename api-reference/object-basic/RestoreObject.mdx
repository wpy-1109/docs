---
title: 'RestoreObject'
description: 'Restore an archived object for reading'
---

Restores an Archive, Cold Archive, or Deep Cold Archive object. After restoration, the object is readable for a specified period.

## Request

```
POST /{ObjectName}?restore HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body (Cold archive / deep cold archive only)

```xml
<?xml version="1.0" encoding="UTF-8"?>
<RestoreRequest>
  <Days>2</Days>
  <JobParameters>
    <Tier>Standard</Tier>
  </JobParameters>
</RestoreRequest>
```

| Element | Type | Description |
|---------|------|-------------|
| `Days` | Integer | Number of days to keep the restored copy (1-7). |
| `Tier` | String | Retrieval speed: `Expedited` (1-5 min), `Standard` (1-5 hours), `Bulk` (5-12 hours). |

### Restore times by storage class

| Storage Class | Restore Time |
|---------------|-------------|
| Archive | About 1 minute |
| Cold Archive - Expedited | 1-5 minutes |
| Cold Archive - Standard | 1-5 hours |
| Cold Archive - Bulk | 5-12 hours |
| Deep Cold Archive - Standard | 5-12 hours |
| Deep Cold Archive - Bulk | 12-48 hours |

## Response

- **HTTP 202**: Restore request accepted, processing.
- **HTTP 200**: Object is already restored.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `InvalidObjectState` | 403 | The object is not in an archival storage class. |
| `RestoreAlreadyInProgress` | 409 | A restore is already in progress. |
