---
title: 'Rename'
description: 'Rename an object within a bucket'
---

Renames an object within the same bucket. Only available for buckets with hierarchical namespace enabled.

## Request

```
POST /{DestObjectName}?rename HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
x-oss-rename-source: {SourceObjectName}
```

### Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `x-oss-rename-source` | String | Yes | Source object key. |

## Response

**Success:** HTTP 200 with empty body.

<Note>
This API is only available for buckets with hierarchical namespace enabled.
</Note>
