---
title: 'PutBucketRequestPayment'
description: 'Enable or disable pay-by-requester for a bucket'
---

Sets a bucket to pay-by-requester mode. When enabled, requesters (not the bucket owner) pay for requests and data transfer.

## Request

```
PUT /?requestPayment HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<RequestPaymentConfiguration>
  <Payer>Requester</Payer>
</RequestPaymentConfiguration>
```

| Element | Type | Description |
|---------|------|-------------|
| `Payer` | String | `BucketOwner` (default) or `Requester`. |

## Response

**Success:** HTTP 200 with empty body.

<Note>
Requesters must include `x-oss-request-payer: requester` in their requests to access a pay-by-requester bucket.
</Note>
