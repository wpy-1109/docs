---
title: 'PutSymlink'
description: 'Create a symbolic link to an object'
---

Creates a symbolic link (symlink) that points to an existing object in the same bucket.

## Request

```
PUT /{SymlinkName}?symlink HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
x-oss-symlink-target: {TargetObjectName}
```

### Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `x-oss-symlink-target` | String | Yes | Target object key. Must be URL-encoded. |
| `x-oss-storage-class` | String | No | Storage class for the symlink object. |
| `x-oss-object-acl` | String | No | ACL for the symlink object. |

## Response

**Success:** HTTP 200 with empty body.

<Note>
The target object does not need to exist when creating the symlink. A `NoSuchKey` error occurs when reading the symlink if the target does not exist.
</Note>
