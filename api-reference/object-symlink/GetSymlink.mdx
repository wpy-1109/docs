---
title: 'GetSymlink'
description: 'Query the target of a symbolic link'
---

Queries the target object that a symlink points to.

## Request

```
GET /{SymlinkName}?symlink HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```http
HTTP/1.1 200 OK
x-oss-symlink-target: target-object.txt
ETag: "D41D8CD98F00B204E9800998ECF8****"
```
