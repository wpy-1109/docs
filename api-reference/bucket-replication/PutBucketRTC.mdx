---
title: 'PutBucketRTC'
description: 'Enable or disable replication time control (RTC)'
---

Enables or disables replication time control (RTC) for existing cross-region replication rules. RTC guarantees that most objects are replicated within minutes.

## Request

```
PUT /?rtc HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ReplicationRule>
  <ID>rule-1</ID>
  <RTC>
    <Status>enabled</Status>
  </RTC>
</ReplicationRule>
```

## Response

**Success:** HTTP 200 with empty body.

<Note>
RTC incurs additional fees. It ensures that 99.99% of objects are replicated within 10 minutes.
</Note>
