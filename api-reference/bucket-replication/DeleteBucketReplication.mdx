---
title: 'DeleteBucketReplication'
description: 'Stop replication and delete the replication configuration'
---

Stops data replication tasks and deletes the replication configuration of a bucket.

## Request

```
POST /?replicationDelete HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ReplicationRules>
  <ID>rule-1</ID>
</ReplicationRules>
```

## Response

**Success:** HTTP 200 with empty body.
