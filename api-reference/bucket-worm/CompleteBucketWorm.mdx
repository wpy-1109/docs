---
title: 'CompleteBucketWorm'
description: 'Lock a retention policy'
---

Locks a WORM retention policy. Once locked, the policy cannot be deleted or shortened.

## Request

```
POST /?wormId={WormId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `wormId` | String | Yes | WORM ID returned by InitiateBucketWorm. |

## Response

**Success:** HTTP 200 with empty body.

<Warning>
Locking a WORM policy is irreversible. Objects in the bucket cannot be deleted or overwritten during the retention period.
</Warning>
