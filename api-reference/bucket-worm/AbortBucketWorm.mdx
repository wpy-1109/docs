---
title: 'AbortBucketWorm'
description: 'Delete an unlocked retention policy'
---

Deletes an unlocked WORM retention policy. Locked policies cannot be deleted.

## Request

```
DELETE /?worm HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.
