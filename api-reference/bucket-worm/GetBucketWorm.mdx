---
title: 'GetBucketWorm'
description: 'Query the retention policy of a bucket'
---

Queries the WORM retention policy of a bucket.

## Request

```
GET /?worm HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<WormConfiguration>
  <WormId>1666E2CFB2B3418****</WormId>
  <State>Locked</State>
  <RetentionPeriodInDays>365</RetentionPeriodInDays>
  <CreationDate>2025-01-15T08:30:00.000Z</CreationDate>
</WormConfiguration>
```

| Element | Type | Description |
|---------|------|-------------|
| `WormId` | String | WORM policy ID. |
| `State` | String | `InProgress` (unlocked) or `Locked`. |
| `RetentionPeriodInDays` | Integer | Retention period in days. |
