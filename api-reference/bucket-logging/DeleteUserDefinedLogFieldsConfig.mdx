---
title: 'DeleteUserDefinedLogFieldsConfig'
description: 'Delete custom log fields configuration'
---

Deletes the custom log fields configuration of a bucket.

## Request

```
DELETE /?userDefinedLogFieldsConfig HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.
