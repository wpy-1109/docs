---
title: 'PutBucketHttpsConfig'
description: 'Configure TLS version settings for a bucket'
---

Enables or disables TLS version settings for a bucket to enforce minimum TLS versions.

## Request

```
PUT /?httpsConfig HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<HttpsConfiguration>
  <TLS>
    <Enable>true</Enable>
    <TLSVersion>TLSv1.2</TLSVersion>
  </TLS>
</HttpsConfiguration>
```

| Element | Type | Required | Description |
|---------|------|----------|-------------|
| `Enable` | Boolean | Yes | Enable or disable TLS version enforcement. |
| `TLSVersion` | String | Yes | Minimum TLS version: `TLSv1.0`, `TLSv1.1`, `TLSv1.2`, or `TLSv1.3`. |

## Response

**Success:** HTTP 200 with empty body.
