---
title: 'GetBucketHttpsConfig'
description: 'Query the TLS version settings of a bucket'
---

Queries the TLS version settings of a bucket.

## Request

```
GET /?httpsConfig HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<HttpsConfiguration>
  <TLS>
    <Enable>true</Enable>
    <TLSVersion>TLSv1.2</TLSVersion>
  </TLS>
</HttpsConfiguration>
```
