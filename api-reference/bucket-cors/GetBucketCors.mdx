---
title: 'GetBucketCors'
description: 'Query CORS rules for a bucket'
---

Queries the CORS rules configured for a bucket.

## Request

```
GET /?cors HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

Returns the CORS configuration XML.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchCORSConfiguration` | 404 | No CORS rules are configured. |
