---
title: 'PutCname'
description: 'Bind a custom domain name (CNAME) to a bucket'
---

Binds a CNAME record (custom domain name) to a bucket.

## Request

```
POST /?cname&comp=add HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<BucketCnameConfiguration>
  <Cname>
    <Domain>cdn.example.com</Domain>
  </Cname>
</BucketCnameConfiguration>
```

## Response

**Success:** HTTP 200 with empty body.

<Note>
Before binding a CNAME, you must verify domain ownership using CreateCnameToken and add a TXT record to your DNS.
</Note>
