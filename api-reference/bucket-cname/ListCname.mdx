---
title: 'ListCname'
description: 'List all CNAME records bound to a bucket'
---


## Request

```
GET /?cname HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ListCnameResult>
  <Bucket>my-bucket</Bucket>
  <Owner>51264</Owner>
  <Cname>
    <Domain>cdn.example.com</Domain>
    <Status>Enabled</Status>
    <LastModified>2025-04-17T03:15:40.000Z</LastModified>
  </Cname>
</ListCnameResult>
```
