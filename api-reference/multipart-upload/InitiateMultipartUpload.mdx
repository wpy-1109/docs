---
title: 'InitiateMultipartUpload'
description: 'Initiate a multipart upload and get an upload ID'
---

Initiates a multipart upload and returns an upload ID. Use this ID for subsequent UploadPart and CompleteMultipartUpload requests.

## Request

```
POST /{ObjectName}?uploads HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `Content-Type` | String | No | MIME type for the final object. |
| `x-oss-storage-class` | String | No | Storage class for the object. |
| `x-oss-server-side-encryption` | String | No | Encryption algorithm: `AES256`, `KMS`, `SM4`. |
| `x-oss-forbid-overwrite` | String | No | Set to `true` to prevent overwriting. |
| `x-oss-tagging` | String | No | Object tags. |
| `x-oss-meta-*` | String | No | Custom metadata. |
| `Cache-Control` | String | No | Caching behavior. |
| `Content-Disposition` | String | No | Display/download behavior. |
| `Content-Encoding` | String | No | Content encoding. |
| `Expires` | String | No | Expiration time. |

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<InitiateMultipartUploadResult>
  <Bucket>my-bucket</Bucket>
  <Key>large-file.bin</Key>
  <UploadId>0004B9895DBBB6EC98E36****</UploadId>
</InitiateMultipartUploadResult>
```

| Element | Type | Description |
|---------|------|-------------|
| `UploadId` | String | Upload ID for subsequent part uploads. Valid for a limited time. |

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
