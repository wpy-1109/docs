---
title: 'ListParts'
description: 'List all uploaded parts of a multipart upload'
---


## Request

```
GET /{ObjectName}?uploadId={UploadId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `uploadId` | String | Yes | Upload ID. |
| `max-parts` | Integer | No | Maximum parts to return. Default: 1000. |
| `part-number-marker` | Integer | No | List parts after this number. |

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ListPartsResult>
  <Bucket>my-bucket</Bucket>
  <Key>large-file.bin</Key>
  <UploadId>0004B9895DBBB6EC98E36****</UploadId>
  <IsTruncated>false</IsTruncated>
  <Part>
    <PartNumber>1</PartNumber>
    <LastModified>2025-04-17T03:15:40.000Z</LastModified>
    <ETag>"7265F4D211B56873A381D321F586E4A9"</ETag>
    <Size>10485760</Size>
  </Part>
</ListPartsResult>
```
