---
title: 'AbortMultipartUpload'
description: 'Abort a multipart upload and delete uploaded parts'
---


## Request

```
DELETE /{ObjectName}?uploadId={UploadId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.

<Note>
After aborting, the upload ID becomes invalid. Any subsequent UploadPart requests with this ID will fail.
</Note>
