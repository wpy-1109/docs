---
title: 'GetBucketTransferAcceleration'
description: 'Query the transfer acceleration configuration of a bucket'
---

Queries the transfer acceleration configuration of a bucket.

## Request

```
GET /?transferAcceleration HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<TransferAccelerationConfiguration>
  <Enabled>true</Enabled>
</TransferAccelerationConfiguration>
```
