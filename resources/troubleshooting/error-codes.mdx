## Error response format

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Error>
  <Code>NoSuchBucket</Code>
  <Message>The specified bucket does not exist.</Message>
  <RequestId>5C3D9175B6FC201293AD****</RequestId>
  <HostId>examplebucket.oss-cn-hangzhou.aliyuncs.com</HostId>
</Error>
```

Always include the `RequestId` when contacting support.

## 4xx client errors

| HTTP | Error Code | Description | Solution |
|------|------------|-------------|----------|
| 400 | `InvalidBucketName` | Invalid bucket name | Use 3-63 chars, lowercase, numbers, hyphens |
| 400 | `InvalidObjectName` | Invalid object key | Max 1023 bytes, use URL encoding for special chars |
| 400 | `TooManyBuckets` | Exceeded bucket limit | Request quota increase or delete unused buckets |
| 400 | `InvalidArgument` | Invalid parameter value | Check parameter format and allowed values |
| 400 | `MalformedXML` | XML body is malformed | Validate XML syntax |
| 400 | `InvalidDigest` | Content-MD5 mismatch | Recalculate MD5 hash |
| 400 | `IncompleteBody` | Body size mismatch | Retry, check network stability |
| 400 | `FileImmutable` | WORM-protected object | Cannot modify during retention |
| 400 | `EntityTooLarge` | Exceeds 5 GB single upload | Use multipart upload |
| 400 | `EntityTooSmall` | Part too small | Min 100 KB (except last part) |
| 400 | `InvalidEncryptionAlgorithmError` | Bad encryption algorithm | Use AES256, KMS, or SM4 |
| 400 | `MissingContentLength` | Missing Content-Length | Add header or use chunked encoding |
| 400 | `RequestTimeout` | Body not sent in time | Retry with stable connection |
| 403 | `AccessDenied` | Authorization failed | Check RAM policy, bucket policy, ACL |
| 403 | `InvalidAccessKeyId` | Bad AccessKey ID | Verify key is correct and active |
| 403 | `SignatureDoesNotMatch` | Signature mismatch | Check signing process |
| 403 | `RequestTimeTooSkewed` | Clock off by >15 min | Sync clock with NTP |
| 403 | `InvalidObjectState` | Wrong object state | Restore archived objects first |
| 404 | `NoSuchBucket` | Bucket not found | Verify name and region |
| 404 | `NoSuchKey` | Object not found | Verify key, check lifecycle rules |
| 404 | `NoSuchUpload` | Upload ID not found | May have been aborted/completed |
| 405 | `MethodNotAllowed` | Method not supported | Check allowed methods |
| 409 | `BucketAlreadyExists` | Bucket name taken | Choose a different name |
| 409 | `BucketNotEmpty` | Bucket has objects | Delete all objects first |
| 409 | `ObjectNotAppendable` | Not an appendable object | Use AppendObject to create |
| 409 | `PositionNotEqualToLength` | Append position wrong | Use current object size |
| 412 | `PreconditionFailed` | Conditional check failed | Verify If-Match/If-Unmodified-Since |
| 416 | `InvalidRange` | Bad range request | Check Range header values |

## 5xx server errors

| HTTP | Error Code | Description | Solution |
|------|------------|-------------|----------|
| 500 | `InternalError` | Server error | Retry with exponential backoff |
| 503 | `ServiceUnavailable` | Temporarily unavailable | Retry with exponential backoff |
| 503 | `SlowDown` | Rate limit exceeded | Reduce request rate, add backoff |
| 503 | `Throttling` | Request throttled | Distribute across buckets/prefixes |

## Retry strategy

```python
import time, random

def call_with_retry(func, max_retries=5):
    for attempt in range(max_retries):
        try:
            return func()
        except ServerError as e:
            if e.status < 500 or attempt == max_retries - 1:
                raise
            wait = min(2 ** attempt + random.uniform(0, 1), 30)
            time.sleep(wait)
```

<Warning>
Do not retry 4xx errors without fixing the underlying issue.
</Warning>
