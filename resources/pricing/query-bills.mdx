## Console

1. Log in to the [Alibaba Cloud Billing Management](https://usercenter2.aliyun.com/) console
2. Navigate to **Spending Summary** > **Spending Summary**
3. Filter by product: **Object Storage Service**
4. View breakdown by billing item (storage, traffic, requests)

## API

Use the Alibaba Cloud Billing API to programmatically query costs:

```python
from alibabacloud_bssopenapi20171214 import Client, models

client = Client(config)
request = models.QueryInstanceBillRequest(
    billing_cycle='2025-04',
    product_code='oss'
)
response = client.query_instance_bill(request)
```

## Cost allocation with tags

Use bucket tags to allocate costs across teams or projects:

1. Tag buckets with team, project, or environment identifiers
2. View cost allocation reports filtered by tags
3. Set budget alerts per tag group

## Budget alerts

Configure budget alerts to get notified before costs exceed your budget:

1. Go to **Billing Management** > **Budgets**
2. Create a new budget for OSS
3. Set threshold amounts and notification channels
4. Receive alerts via email or SMS when costs approach limits
