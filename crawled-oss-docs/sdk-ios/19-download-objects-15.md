# 下载文件（iOS SDK）

 
更新时间: 2025-12-01 09:32:57

 

 

本文档介绍 OSS iOS SDK 下载文件的方式。

OSS iOS SDK 提供了如下几种下载文件的方式：
- [简单下载（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/simple-download#concept-gdb-dhg-4fb)
- [流式下载（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/streaming-download-6#concept-bdd-rhg-4fb)
- [范围下载（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/range-download-7#concept-vyd-5hg-4fb)
- [断点续传下载（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/resumable-download#concept-llq-qpb-3gb)
- [限定条件下载（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/conditional-download-6)

上传过程中，您可以通过[进度条（iOS SDK）](https://help.aliyun.com/zh/oss/developer-reference/download-progress-bars)查看下载进度。
