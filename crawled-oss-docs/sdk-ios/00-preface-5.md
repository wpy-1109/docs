# OSS iOS SDK

 
更新时间: 2025-11-28 14:52:57

 

 

本文档基于OSS iOS SDK 2.10.20编写。

## 前提条件
- 已开通OSS服务。具体操作，请参见[开通OSS服务](https://help.aliyun.com/zh/oss/getting-started/activate-oss#task-njz-hf4-tdb)。
- 已创建AccessKey ID和AccessKey Secret。具体操作，请参见[创建AccessKey](https://help.aliyun.com/zh/ram/user-guide/create-an-accesskey-pair#task-2245479)。
- 使用8.0及以上版本iOS系统。
