# Release Notes - Alibaba Cloud OSS

> Source: https://www.alibabacloud.com/help/en/oss/product-overview/release-notes

## Overview

This page summarizes the key feature releases, updates, and enhancements for Alibaba Cloud Object Storage Service (OSS). For the most current and detailed release notes, visit the official documentation.

## 2025 Highlights

### Vector Buckets (New)

- A new bucket type supporting vector data storage and retrieval capabilities
- Provides a simple, scalable, and cost-effective foundation for AI applications
- Enables vector search and similarity matching directly within OSS

### OSS Browser 2.0

- V2.1.1-beta preview release
- Refreshed browser-based management tool for OSS buckets and objects
- Improved user interface and functionality

### Deep Cold Archive Enhancements

- Continued improvements to the ultra-low-cost archival storage tier
- Optimized restore operations and pricing

### Zone-Redundant Storage (ZRS) Improvements

- Standard (ZRS) offering 99.9999999999% (twelve 9s) of data durability
- Enhanced redundancy by distributing data across multiple availability zones

## 2024-2025 Updates by Category

### Storage Classes and Lifecycle Management

- Enhancements to tiered storage (Standard, IA, Archive, Cold Archive, Deep Cold Archive)
- Improved lifecycle transition rules and cost optimization features
- Better support for automatic storage class transitions

### Data Security and Compliance

- Server-side encryption improvements (SSE-KMS, SSE-OSS)
- Bucket policy and access control enhancements
- Data replication and disaster recovery updates
- Enhanced audit logging capabilities

### Performance and Scalability

- Transfer acceleration improvements
- Enhanced multipart upload and download capabilities
- Optimized large file handling
- JindoFS/JindoSDK performance improvements for data lake workloads

### API and SDK Updates

- Updates across multiple SDKs (Java, Python, Go, Node.js, PHP, .NET, C++, Ruby)
- New API features for data processing and management
- V2 SDK releases with improved interfaces

### Integration and Ecosystem

- Tighter integration with other Alibaba Cloud services (CDN, Function Compute, DataWorks)
- IMG (Image Processing) and media transcoding enhancements
- OSS-HDFS (JindoFS) data lake storage improvements
- Enhanced EMR and MaxCompute integration

### Cost Management

- New billing options and resource pack updates
- Pay-as-you-go refinements
- Storage Capacity Units (SCU) improvements

### Tools and Utilities

- ossutil updates and new features
- ossbrowser improvements
- Console UI/UX enhancements
