# Query bills

This topic describes how to query bills of Object Storage Service (OSS).

## Procedure


-

Log on to the [Expenses and Costs console](https://usercenter2-intl.console.alibabacloud.com/).

-

In the left-side navigation pane, choose Bills > Bill Details.

-

On the Bill Details page, click the Billing Details tab, specify Billing Cycle, and set Product to Object Storage Service.


You can also configure Statistic Item and Statistic Period to specify the content of the bills that you want to display.

| Parameter | Description |
| --- | --- |
| Statistic Item | Bills are displayed based on the statistic item that you specify. Valid values: Billing Item: Bills are displayed by billable item. The bills that are generated based on the same billable item are combined. Instance: Bills are displayed by instance. The bills that are generated by the same instance are combined. Product: Bills are displayed by product. The bills that are generated by the same product are combined. Account: Bills are displayed by account. The bills that are generated in the same account are combined. Cost Center: Bills are displayed by cost center. The bills that are generated by the same cost center are combined. |
| Statistic Period | Bills are displayed based on the statistic period that you specify. Valid values: Billing Cycle: The bills of the specified billing cycle are displayed. By Day: The bills of the specified billing cycle are displayed by day. Billing Period: The bills of the specified billing cycle are displayed by hour. |

> NOTE: Note 

-

OSS is billed on an hourly basis. If you select Billing Cycle or By Day for Statistic Period, data in the Usage column specifies the cumulative usage in the current month or on the current day. For example, you select By Day for Statistic Period and your actual storage usage is 1 TB. The storage usage that is displayed in the bill is 24 TB. You are charged for the 24-hour storage of 1 TB of data.

-

You can click Customize Column Options in the upper-right corner of the bills and select the columns that you want to view.

-

You can click Export Billing Overview (CSV) to export the bills to your computer. This way, you can view and analyze the bills on your computer.
