# Overview of logging and monitoring

Logging and monitoring are essential for ensuring the availability of your Object Storage Service (OSS) resources, maintaining business continuity, and monitoring system health. Alibaba Cloud provides various services for monitoring and log auditing, such as CloudMonitor, Simple Log Service, and Cloud Config. These services help you monitor your cloud resource usage and business status in real time and respond promptly to alerts for abnormal events.


## CloudMonitor


OSS is integrated with CloudMonitor. You can use this free service to monitor various OSS metrics in real time. For more information, see [Monitoring service](https://www.alibabacloud.com/help/en/oss/user-guide/overview-72/).


## Log storage


Accessing OSS generates numerous access logs. You can use the log storage feature to create hourly log files based on a fixed naming convention and write them to a specified bucket. For more information, see [Log storage](https://www.alibabacloud.com/help/en/oss/user-guide/logging).

## Simple Log Service


OSS is integrated with Simple Log Service (SLS). For your OSS buckets, you can enable the real-time logging feature to collect and process log data about cloud products, such as operational information, running status, and business dynamics. SLS analyzes and delivers this data in real time for monitoring and auditing. For more information, see [Real-time log query](https://www.alibabacloud.com/help/en/oss/user-guide/real-time-log-query/).

## Cloud Config


OSS is integrated with Cloud Config and offers a free trial. You can use Cloud Config to track the configuration history and audit the compliance of your cloud resources. This helps you monitor your resources and ensure the ongoing compliance of your infrastructure. For more information, see [Cloud Config for OSS](https://www.alibabacloud.com/help/en/cloud-config/latest/oss).

## ActionTrail


OSS is integrated with ActionTrail. You can use ActionTrail to monitor and record operations performed by users under your Alibaba Cloud account, such as logons and resource access. ActionTrail also provides security analytics, intrusion detection, resource change tracking, and compliance auditing. For more information, see [ActionTrail for OSS](https://www.alibabacloud.com/help/en/actiontrail/user-guide/oss).
