# Analyze and process data in OSS

Object Storage Service (OSS) offers over 20 data processing features for resources such as images, documents, media, and files. These features are provided through native OSS capabilities and integration with Intelligent Media Management (IMM).


OSS supports the following data analysis and processing features:











(https://www.alibabacloud.com/help/en/oss/user-guide/overview-17/#concept-2333980)


(https://www.alibabacloud.com/help/en/oss/user-guide/video-snapshots#concept-kz1-cwc-wdb)


(https://www.alibabacloud.com/help/en/oss/user-guide/zip-package-decompression#concept-x1g-cyg-4gb)


(https://www.alibabacloud.com/help/en/oss/user-guide/real-time-processing-file-changes-oss-event-notifications#concept-727727)


| Feature | Description | References |
| --- | --- | --- |
| Image processing | Image processing includes features such as image scaling, custom cropping, and image styling. | Overview of image processing |
| Video snapshots | OSS supports taking snapshots of video files encoded in H.264 and H.265. | Take a single-frame snapshot |
| ZIP package decompression | OSS provides an automatic decompression feature for ZIP files. After you configure decompression rules, ZIP files that you upload to a specified path in OSS are automatically decompressed. | Upload and automatically decompress ZIP packages |
| Event notifications | In the OSS console, set event notification rules for objects that you want to monitor. When a specified event occurs on these objects, you receive a notification. | Process real-time changes to OSS files using event notifications |
