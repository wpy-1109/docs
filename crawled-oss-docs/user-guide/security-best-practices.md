# Security best practices

When utilizing OSS, safeguarding against various threats is crucial to ensure the security and stability of data storage and transmission, thereby maintaining smooth business operations. This topic describes common threats and their countermeasures, including account security, malicious traffic attacks, data loss, sensitive data masking, and Bucket security configuration, offering comprehensive guidance for threat prevention.

## Risk avoidance


In the course of using OSS, you may encounter a variety of security threats, such as password leaks, malicious access, operational errors, and vulnerabilities in Bucket configuration security. Please consult the following documents as needed to understand preventive measures and ensure data security.
