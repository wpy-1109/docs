# Installation

## Prerequisites

- PHP 7.2 or later
- Composer package manager

## Install V2 SDK (Recommended)

```bash
composer require alibabacloud/oss-v2
```

## Install V1 SDK

```bash
composer require aliyuncs/oss-sdk-php
```

## Verify installation

```php title="verify.php"
<?php
require_once __DIR__ . '/vendor/autoload.php';
echo "OSS SDK installed successfully\n";
```
