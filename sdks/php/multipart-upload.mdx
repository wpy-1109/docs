# Multipart upload

## Resumable upload (Recommended)

```php title="resumableUpload.php"
$options = [
    OssClient::OSS_CHECK_MD5 => true,
    OssClient::OSS_PART_SIZE => 10 * 1024 * 1024, // 10MB per part
];
$ossClient->multiuploadFile($bucket, 'large-file.zip', '/path/to/large-file.zip', $options);
```

## Manual multipart upload

```php title="multipartUpload.php"
// Step 1: Initiate
$uploadId = $ossClient->initiateMultipartUpload($bucket, 'large-file.zip');

// Step 2: Upload parts
$partSize = 5 * 1024 * 1024; // 5MB
$fileSize = filesize('/path/to/large-file.zip');
$parts = [];

for ($i = 0; $i * $partSize < $fileSize; $i++) {
    $offset = $i * $partSize;
    $size = min($partSize, $fileSize - $offset);
    $content = file_get_contents('/path/to/large-file.zip', false, null, $offset, $size);

    $result = $ossClient->uploadPart($bucket, 'large-file.zip', $uploadId, [
        'partNumber' => $i + 1,
        'body' => $content,
    ]);
    $parts[] = ['PartNumber' => $i + 1, 'ETag' => $result];
}

// Step 3: Complete
$ossClient->completeMultipartUpload($bucket, 'large-file.zip', $uploadId, $parts);
```
