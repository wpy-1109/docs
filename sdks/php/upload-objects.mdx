# Upload Objects

## Upload a string

```php title="uploadString.php"
$ossClient->putObject($bucket, 'example.txt', 'Hello, OSS!');
```

## Upload a local file

```php title="uploadFile.php"
$ossClient->uploadFile($bucket, 'documents/report.pdf', '/path/to/report.pdf');
```

## Upload with metadata

```php title="uploadMetadata.php"
$options = [
    OssClient::OSS_HEADERS => [
        'Content-Type' => 'text/plain',
        'x-oss-meta-author' => 'your-name',
    ],
];
$ossClient->putObject($bucket, 'example.txt', 'content', $options);
```

## Append upload

```php title="appendUpload.php"
$position = $ossClient->appendObject($bucket, 'append-log.txt', 'First entry\n', 0);
$position = $ossClient->appendObject($bucket, 'append-log.txt', 'Second entry\n', $position);
```
