# Download objects

## Download to memory

```php title="downloadMemory.php"
$content = $ossClient->getObject($bucket, 'example.txt');
echo $content;
```

## Download to a local file

```php title="downloadFile.php"
$options = [OssClient::OSS_FILE_DOWNLOAD => '/path/to/local/file.txt'];
$ossClient->getObject($bucket, 'example.txt', $options);
```

## Range download

```php title="rangeDownload.php"
$options = [OssClient::OSS_RANGE => '0-1023']; // First 1 KB
$content = $ossClient->getObject($bucket, 'example.txt', $options);
```
