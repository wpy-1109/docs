# Bucket operations

## Create a bucket

```php title="createBucket.php"
$ossClient->createBucket($bucket);

// With options
$options = [
    OssClient::OSS_STORAGE => OssClient::OSS_STORAGE_STANDARD,
    OssClient::OSS_ACL => OssClient::OSS_ACL_TYPE_PRIVATE,
];
$ossClient->createBucket($bucket, $options);
```

## List buckets

```php title="listBuckets.php"
$bucketList = $ossClient->listBuckets();
foreach ($bucketList->getBucketList() as $bucket) {
    echo "Bucket: " . $bucket->getName() . ", Location: " . $bucket->getLocation() . "\n";
}
```

## Get bucket info

```php title="getBucketInfo.php"
$info = $ossClient->getBucketInfo($bucket);
echo "Name: " . $info->getName() . "\n";
echo "Location: " . $info->getLocation() . "\n";
echo "StorageClass: " . $info->getStorageClass() . "\n";
```

## Delete a bucket

```php title="deleteBucket.php"
$ossClient->deleteBucket($bucket);
```
