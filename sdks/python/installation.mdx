# Installation

The Alibaba Cloud OSS Python SDK is available in two versions. The V2 SDK is recommended for new projects.

## Prerequisites

- Python 3.x (Python 2.x is no longer supported)
- pip package manager

## SDK versions

| Feature | V1 SDK (`oss2`) | V2 SDK (`alibabacloud-oss-v2`) |
|---------|-----------------|-------------------------------|
| Install command | `pip install oss2` | `pip install alibabacloud-oss-v2` |
| Status | Legacy (maintained) | **Recommended** |
| Async support | No | Yes |
| CRC checksum | Basic | Enhanced |
| Credential providers | Basic | Full credential chain |
| Signature version | V1 | V4 (more secure) |

## Install V2 SDK (Recommended)

```bash
pip install alibabacloud-oss-v2
```

Verify the installation:

```bash
python -c "import alibabacloud_oss_v2; print('OSS SDK V2 installed successfully')"
```

## Install V1 SDK

```bash
pip install oss2
```

Verify the installation:

```bash
python -c "import oss2; print(oss2.__version__)"
```

## Install from source

To install the V2 SDK from source:

```bash
git clone https://github.com/aliyun/alibabacloud-oss-python-sdk-v2.git
cd alibabacloud-oss-python-sdk-v2
pip install -e .
```
