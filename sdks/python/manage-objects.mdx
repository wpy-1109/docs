# Manage Objects

Manage objects in OSS including listing, copying, deleting, and working with metadata, ACLs, and symbolic links.

## List objects

```python title="list_objects.py"
import oss2

# List all objects
for obj in oss2.ObjectIterator(bucket):
    print(f'Key: {obj.key}, Size: {obj.size}')

# List with prefix filter
for obj in oss2.ObjectIterator(bucket, prefix='photos/'):
    print(f'Key: {obj.key}')

# List with delimiter (directory-like listing)
for obj in oss2.ObjectIterator(bucket, prefix='photos/', delimiter='/'):
    print(f'Key: {obj.key}')
```

## Check if an object exists

```python title="object_exists.py"
exists = bucket.object_exists('example.txt')
print(f'Object exists: {exists}')
```

## Copy objects

```python title="copy_objects.py"
# Copy within the same bucket
bucket.copy_object(bucket.bucket_name, 'source.txt', 'destination.txt')

# Copy across buckets
bucket.copy_object('source-bucket', 'source.txt', 'destination.txt')
```

<Note>
For objects larger than 1 GB, use multipart copy instead of simple copy.
</Note>

## Delete objects

```python title="delete_objects.py"
# Delete a single object
bucket.delete_object('example.txt')

# Delete multiple objects (batch delete, up to 1000 per request)
bucket.batch_delete_objects(['key1.txt', 'key2.txt', 'key3.txt'])
```

## Object metadata

### Get object metadata

```python title="get_metadata.py"
# Basic metadata (lightweight, returns only size and ETag)
meta = bucket.get_object_meta('example.txt')
print(f'Content-Length: {meta.headers["Content-Length"]}')
print(f'ETag: {meta.headers["ETag"]}')

# Full metadata (includes Content-Type, Last-Modified, custom metadata)
head = bucket.head_object('example.txt')
print(f'Content-Type: {head.content_type}')
print(f'Last-Modified: {head.last_modified}')
```

### Set object metadata on upload

```python title="set_metadata.py"
headers = {
    'Content-Type': 'image/jpeg',
    'x-oss-meta-photographer': 'John Doe',
    'x-oss-meta-location': 'Beijing',
}
bucket.put_object('photo.jpg', data, headers=headers)
```

## Object ACL

```python title="object_acl.py"
# Set object ACL
bucket.put_object_acl('example.txt', oss2.OBJECT_ACL_PUBLIC_READ)

# Get object ACL
acl = bucket.get_object_acl('example.txt')
print(f'Object ACL: {acl.acl}')
```

| ACL Value | Description |
|-----------|-------------|
| `OBJECT_ACL_DEFAULT` | Inherits the bucket ACL |
| `OBJECT_ACL_PRIVATE` | Only the owner can read and write |
| `OBJECT_ACL_PUBLIC_READ` | Anyone can read; only owner can write |
| `OBJECT_ACL_PUBLIC_READ_WRITE` | Anyone can read and write |

## Symbolic links

```python title="symlinks.py"
# Create a symbolic link
bucket.put_symlink('my-symlink', 'target-object.txt')

# Get the target of a symbolic link
result = bucket.get_symlink('my-symlink')
print(f'Target: {result.target_key}')
```
