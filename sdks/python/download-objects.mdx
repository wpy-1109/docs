# Download Objects

Download objects from OSS to memory, to a local file, or as a stream.

## Simple download

### Download to memory

<Tabs>
  <Tab title="V2 SDK">
    ```python title="download_memory_v2.py"
    import alibabacloud_oss_v2 as oss

    result = client.get_object(oss.GetObjectRequest(
        bucket='my-bucket',
        key='example.txt',
    ))
    content = result.body.read()
    print(f'Content: {content.decode()}')
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```python title="download_memory_v1.py"
    result = bucket.get_object('example.txt')
    content = result.read().decode('utf-8')
    print(f'Content: {content}')
    ```
  </Tab>
</Tabs>

### Download to a local file

```python title="download_file.py"
bucket.get_object_to_file('example.txt', '/path/to/local/file.txt')
```

## Range download

Download only a specific byte range of an object:

```python title="range_download.py"
# Download the first 1 KB
result = bucket.get_object('example.txt', byte_range=(0, 1023))
content = result.read()
print(f'Downloaded {len(content)} bytes')
```

## Conditional download

Download an object only if certain conditions are met:

```python title="conditional_download.py"
# Download only if modified after a specific date
result = bucket.get_object('example.txt',
    headers={'If-Modified-Since': 'Wed, 01 Jan 2025 00:00:00 GMT'})

# Download only if ETag matches
result = bucket.get_object('example.txt',
    headers={'If-Match': '"expected-etag-value"'})
```

| Condition Header | Behavior |
|-----------------|----------|
| `If-Modified-Since` | Download only if modified after the given date |
| `If-Unmodified-Since` | Download only if NOT modified after the given date |
| `If-Match` | Download only if the ETag matches |
| `If-None-Match` | Download only if the ETag does NOT match |

## Resumable download

For large files, use resumable download to handle interruptions:

```python title="resumable_download.py"
import oss2

oss2.resumable_download(
    bucket,
    'large-files/video.mp4',
    '/path/to/local/video.mp4',
    store=oss2.ResumableDownloadStore(root='/tmp'),
    multiget_threshold=100 * 1024 * 1024,  # Use multi-range for files > 100MB
    part_size=10 * 1024 * 1024,             # 10MB per part
    num_threads=4                            # 4 concurrent download threads
)
```

## Download with progress tracking

```python title="download_progress.py"
import sys

def progress_callback(consumed_bytes, total_bytes):
    if total_bytes:
        rate = 100 * (float(consumed_bytes) / float(total_bytes))
        sys.stdout.write(f'\rDownloading: {rate:.1f}%')
        sys.stdout.flush()

bucket.get_object_to_file(
    'large-file.zip',
    '/path/to/local/file.zip',
    progress_callback=progress_callback
)
print('\nDownload complete')
```
