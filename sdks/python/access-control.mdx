# Access Control

Manage access to your OSS resources using ACLs, bucket policies, CORS rules, and STS temporary credentials.

## Bucket ACL

```python title="bucket_acl.py"
import oss2

# Set bucket ACL
bucket.put_bucket_acl(oss2.BUCKET_ACL_PRIVATE)

# Get bucket ACL
acl = bucket.get_bucket_acl()
print(f'Bucket ACL: {acl.acl}')
```

| ACL | Description |
|-----|-------------|
| `BUCKET_ACL_PRIVATE` | Only the owner can read and write |
| `BUCKET_ACL_PUBLIC_READ` | Anyone can read; only owner can write |
| `BUCKET_ACL_PUBLIC_READ_WRITE` | Anyone can read and write |

## Object ACL

```python title="object_acl.py"
# Set object ACL
bucket.put_object_acl('example.txt', oss2.OBJECT_ACL_PUBLIC_READ)

# Get object ACL
acl = bucket.get_object_acl('example.txt')
print(f'Object ACL: {acl.acl}')
```

| ACL | Description |
|-----|-------------|
| `OBJECT_ACL_DEFAULT` | Inherits the bucket ACL |
| `OBJECT_ACL_PRIVATE` | Only the owner can read and write |
| `OBJECT_ACL_PUBLIC_READ` | Anyone can read; only owner can write |
| `OBJECT_ACL_PUBLIC_READ_WRITE` | Anyone can read and write |

## CORS configuration

Configure Cross-Origin Resource Sharing for browser-based access:

```python title="cors.py"
from oss2.models import CorsRule, BucketCors

rule = CorsRule(
    allowed_origins=['https://example.com', 'https://app.example.com'],
    allowed_methods=['GET', 'PUT', 'POST', 'DELETE'],
    allowed_headers=['*'],
    expose_headers=['ETag', 'x-oss-request-id'],
    max_age_seconds=3600
)
bucket.put_bucket_cors(BucketCors([rule]))

# Get CORS rules
cors = bucket.get_bucket_cors()
for rule in cors.rules:
    print(f'Origins: {rule.allowed_origins}')
    print(f'Methods: {rule.allowed_methods}')

# Delete CORS rules
bucket.delete_bucket_cors()
```

## STS temporary credentials

Use Security Token Service (STS) for temporary, scoped access:

```python title="sts_auth.py"
import oss2

auth = oss2.StsAuth(
    'temporary-access-key-id',
    'temporary-access-key-secret',
    'security-token'
)
bucket = oss2.Bucket(auth, 'https://oss-cn-hangzhou.aliyuncs.com', 'my-bucket')

# Now perform operations with temporary credentials
bucket.put_object('example.txt', 'content')
```

<Tip>
STS credentials are ideal for granting temporary access to mobile apps, browser clients, or third-party services. Set the shortest session duration that meets your needs (typically 15 minutes to 1 hour).
</Tip>
