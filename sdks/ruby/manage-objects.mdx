# Manage Objects

## List objects

```ruby title="listObjects.rb"
require 'aliyun/oss'

client = Aliyun::OSS::Client.new(
  endpoint: 'https://oss-cn-hangzhou.aliyuncs.com',
  access_key_id: ENV['OSS_ACCESS_KEY_ID'],
  access_key_secret: ENV['OSS_ACCESS_KEY_SECRET']
)

bucket = client.get_bucket('my-bucket')

# List all objects
objects = bucket.list_objects
objects.each { |o| puts o.key }
```

## List objects by prefix

```ruby title="listByPrefix.rb"
objects = bucket.list_objects(:prefix => 'photos/')
objects.each { |o| puts o.key }
```

## List with marker (Pagination)

```ruby title="listWithMarker.rb"
objects = bucket.list_objects(:prefix => 'logs/', :marker => 'logs/2024-01')
objects.each { |o| puts o.key }
```

## List directory contents

Simulate folder listing using the delimiter parameter:

```ruby title="listDirectory.rb"
def list_dir(dir, bucket)
  objects = bucket.list_objects(:prefix => dir, :delimiter => '/')
  objects.each do |obj|
    if obj.is_a?(Aliyun::OSS::Object)
      puts "Object: #{obj.key}"
    else
      puts "SubDir: #{obj}"
      list_dir(obj, bucket)
    end
  end
end

list_dir('foo/', bucket)
```

## Copy an object

```ruby title="copyObject.rb"
# Copy with original metadata
bucket.copy_object(
  'dest-key.txt', 'source-key.txt',
  :meta_directive => Aliyun::OSS::MetaDirective::COPY
)

# Copy with new metadata
bucket.copy_object(
  'dest-key.txt', 'source-key.txt',
  :metas => { 'year' => '2025' },
  :meta_directive => Aliyun::OSS::MetaDirective::REPLACE
)
```

## Delete an object

```ruby title="deleteObject.rb"
bucket.delete_object('old-file.txt')
```

## Batch delete objects

```ruby title="batchDelete.rb"
keys = ['file-1.txt', 'file-2.txt', 'file-3.txt']

# Returns deleted keys by default
result = bucket.batch_delete_objects(keys)
puts result  # ['file-1.txt', 'file-2.txt', 'file-3.txt']

# Quiet mode - no response
bucket.batch_delete_objects(keys, :quiet => true)
```

## Error handling

```ruby title="errorHandling.rb"
begin
  bucket.delete_object('protected-file.txt')
rescue Aliyun::OSS::ServerError => e
  puts "Error Code: #{e.error_code}"
  puts "Message: #{e.message}"
  puts "Request ID: #{e.request_id}"
  puts "HTTP Status: #{e.http_code}"
rescue Aliyun::OSS::ClientError => e
  puts "Client Error: #{e.message}"
end
```
