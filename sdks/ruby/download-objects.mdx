# Download Objects

## Download to a local file

```ruby title="downloadToFile.rb"
require 'aliyun/oss'

client = Aliyun::OSS::Client.new(
  endpoint: 'https://oss-cn-hangzhou.aliyuncs.com',
  access_key_id: ENV['OSS_ACCESS_KEY_ID'],
  access_key_secret: ENV['OSS_ACCESS_KEY_SECRET']
)

bucket = client.get_bucket('my-bucket')
bucket.get_object('documents/report.pdf', :file => '/path/to/local/report.pdf')
```

## Streaming download

Process data in chunks without loading the entire file into memory:

```ruby title="streamingDownload.rb"
bucket.get_object('documents/report.pdf') do |chunk|
  puts "Got a chunk, size: #{chunk.size}."
end
```

## Resumable download

For large files, use resumable download with automatic checkpointing:

```ruby title="resumableDownload.rb"
bucket.resumable_download(
  'videos/large-video.mp4',
  '/path/to/local/large-video.mp4'
) do |p|
  puts "Progress: #{p}"
end
```

With custom part size and checkpoint file:

```ruby title="resumableDownloadCustom.rb"
bucket.resumable_download(
  'videos/large-video.mp4',
  '/path/to/local/large-video.mp4',
  :part_size => 100 * 1024,
  :cpt_file => '/tmp/large-video.mp4.cpt'
) { |p| puts "Progress: #{p}" }
```

### Resumable download parameters

| Parameter | Description | Default |
| --- | --- | --- |
| `key` | Object key in OSS | Required |
| `file` | Local file path to save to | Required |
| `:part_size` | Size of each part (bytes) | 10 MB |
| `:cpt_file` | Checkpoint file path | `file.cpt` |
| `:disable_cpt` | Disable checkpointing | `false` |
