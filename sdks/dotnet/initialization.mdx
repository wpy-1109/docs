# Initialization

## Create an OSS client

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    The V2 SDK uses `AlibabaCloud.OSS.V2.Client` with a configuration object:

    ```csharp title="initV2.cs"
    using OSS = AlibabaCloud.OSS.V2;

    // Load default configuration (reads credentials from environment variables)
    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    // Create the OSS client
    using var client = new OSS.Client(cfg);
    ```

    To specify a custom endpoint:

    ```csharp title="initV2Custom.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";
    cfg.Endpoint = "https://oss-cn-hangzhou.aliyuncs.com";

    using var client = new OSS.Client(cfg);
    ```
  </Tab>
  <Tab title="V1 SDK">
    The V1 SDK uses `Aliyun.OSS.OssClient`:

    ```csharp title="initV1.cs"
    using Aliyun.OSS;
    using Aliyun.OSS.Common;

    var accessKeyId = Environment.GetEnvironmentVariable("OSS_ACCESS_KEY_ID");
    var accessKeySecret = Environment.GetEnvironmentVariable("OSS_ACCESS_KEY_SECRET");
    const string endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    const string region = "cn-hangzhou";

    // Use V4 signature (recommended, requires SDK 2.14.0+)
    var conf = new ClientConfiguration();
    conf.SignatureVersion = SignatureVersion.V4;

    var client = new OssClient(endpoint, accessKeyId, accessKeySecret, conf);
    client.SetRegion(region);
    ```
  </Tab>
</Tabs>

## Credential providers

<Tabs>
  <Tab title="Environment Variables">
    The recommended approach for most deployments. Set `OSS_ACCESS_KEY_ID` and `OSS_ACCESS_KEY_SECRET` as environment variables.

    ```csharp title="envCredentials.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);
    ```
  </Tab>
  <Tab title="STS Token">
    Use temporary security credentials from STS for time-limited access.

    ```csharp title="stsCredentials.cs"
    using Aliyun.OSS;
    using Aliyun.OSS.Common;
    using Aliyun.OSS.Common.Authentication;

    var accessKeyId = Environment.GetEnvironmentVariable("ALIBABA_CLOUD_ACCESS_KEY_ID");
    var accessKeySecret = Environment.GetEnvironmentVariable("ALIBABA_CLOUD_ACCESS_KEY_SECRET");
    var token = Environment.GetEnvironmentVariable("ALIBABA_CLOUD_SECURITY_TOKEN");

    var credentialsProvider = new DefaultCredentialsProvider(
        new DefaultCredentials(accessKeyId, accessKeySecret, token)
    );

    const string endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    const string region = "cn-hangzhou";

    var conf = new ClientConfiguration();
    var client = new OssClient(endpoint, credentialsProvider, conf);
    client.SetRegion(region);
    ```
  </Tab>
  <Tab title="ECS RAM Role">
    For applications running on ECS instances. Credentials are automatically obtained and refreshed.

    ```csharp title="ecsRamRole.cs"
    using Aliyun.OSS;
    using Aliyun.OSS.Common;
    using Aliyun.OSS.Common.Authentication;

    class CredentialsProviderWrapper : ICredentialsProvider
    {
        private Aliyun.Credentials.Client client;
        public CredentialsProviderWrapper(Aliyun.Credentials.Client client)
        {
            this.client = client;
        }
        public ICredentials GetCredentials()
        {
            return new DefaultCredentials(
                client.GetAccessKeyId(),
                client.GetAccessKeySecret(),
                client.GetSecurityToken()
            );
        }
        public void SetCredentials(ICredentials creds) {}
    }

    var config = new Aliyun.Credentials.Models.Config()
    {
        Type = "ecs_ram_role",
        RoleName = "your-ecs-ram-role"
    };
    var credentialsClient = new Aliyun.Credentials.Client(config);
    var credentialsProvider = new CredentialsProviderWrapper(credentialsClient);

    const string endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
    const string region = "cn-hangzhou";

    var conf = new ClientConfiguration();
    var client = new OssClient(endpoint, credentialsProvider, conf);
    client.SetRegion(region);
    ```
  </Tab>
</Tabs>

## Client configuration (V1)

| Parameter | Description | Default |
|-----------|-------------|---------|
| `ConnectionLimit` | Maximum concurrent connections | 512 |
| `MaxErrorRetry` | Maximum retry count on failure | 3 |
| `ConnectionTimeout` | Connection timeout in milliseconds | -1 (no timeout) |
| `EnalbeMD5Check` | Enable MD5 verification on upload/download | `false` |
| `IsCname` | Whether endpoint is a custom domain (CNAME) | `false` |
| `SignatureVersion` | Set to `SignatureVersion.V4` for V4 signatures | V1 |

```csharp title="clientConfig.cs"
using Aliyun.OSS.Common;

var conf = new ClientConfiguration();
conf.ConnectionLimit = 512;
conf.MaxErrorRetry = 3;
conf.ConnectionTimeout = 300;
conf.EnalbeMD5Check = true;
conf.SignatureVersion = SignatureVersion.V4;
```
