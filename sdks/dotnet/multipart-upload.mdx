# Multipart upload

Multipart upload splits large files into smaller parts for parallel uploading. This is recommended for files larger than 100 MB.

## Multipart upload flow

<Steps>
  <Step title="Initiate">
    Call `InitiateMultipartUploadAsync` to get an `UploadId`.
  </Step>
  <Step title="Upload Parts">
    Split the file and upload each part with `UploadPartAsync`.
  </Step>
  <Step title="Complete">
    Call `CompleteMultipartUploadAsync` to assemble all parts.
  </Step>
</Steps>

## Complete example

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="multipartUpload.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var bucket = "my-bucket";
    var key = "large-file.zip";
    var filePath = "/path/to/large-file.zip";
    var partSize = 5 * 1024 * 1024; // 5 MB per part

    // Step 1: Initiate multipart upload
    var initResult = await client.InitiateMultipartUploadAsync(new()
    {
        Bucket = bucket,
        Key = key
    });

    // Step 2: Upload parts
    using var file = File.OpenRead(filePath);
    long fileSize = file.Length;
    int partNumber = 1;
    var uploadParts = new List<OSS.Models.UploadPart>();

    for (long offset = 0; offset < fileSize; offset += partSize)
    {
        var currentPartSize = (int)Math.Min(partSize, fileSize - offset);
        var buffer = new byte[currentPartSize];
        await file.ReadAsync(buffer, 0, currentPartSize);

        var uploadResult = await client.UploadPartAsync(new OSS.Models.UploadPartRequest()
        {
            Bucket = bucket,
            Key = key,
            UploadId = initResult.UploadId,
            PartNumber = partNumber,
            Body = new MemoryStream(buffer)
        });

        uploadParts.Add(new OSS.Models.UploadPart()
        {
            PartNumber = partNumber,
            ETag = uploadResult.ETag
        });

        Console.WriteLine($"Part {partNumber} uploaded ({currentPartSize} bytes)");
        partNumber++;
    }

    // Step 3: Complete multipart upload
    uploadParts.Sort((a, b) => a.PartNumber.CompareTo(b.PartNumber));

    var completeResult = await client.CompleteMultipartUploadAsync(new()
    {
        Bucket = bucket,
        Key = key,
        UploadId = initResult.UploadId,
        CompleteMultipartUpload = new()
        {
            Parts = uploadParts
        }
    });

    Console.WriteLine($"Multipart upload complete. Status: {completeResult.StatusCode}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="multipartUploadV1.cs"
    using Aliyun.OSS;

    var bucket = "my-bucket";
    var key = "large-file.zip";
    var filePath = "/path/to/large-file.zip";
    var partSize = 5 * 1024 * 1024L;

    // Step 1: Initiate
    var initRequest = new InitiateMultipartUploadRequest(bucket, key);
    var initResult = ossClient.InitiateMultipartUpload(initRequest);

    // Step 2: Upload parts
    var partETags = new List<PartETag>();
    using var fileStream = File.OpenRead(filePath);
    long fileSize = fileStream.Length;
    int partNumber = 1;

    for (long offset = 0; offset < fileSize; offset += partSize)
    {
        var size = Math.Min(partSize, fileSize - offset);
        var request = new UploadPartRequest(bucket, key, initResult.UploadId)
        {
            PartNumber = partNumber,
            InputStream = fileStream,
            PartSize = size
        };

        var uploadResult = ossClient.UploadPart(request);
        partETags.Add(uploadResult.PartETag);
        partNumber++;
    }

    // Step 3: Complete
    var completeRequest = new CompleteMultipartUploadRequest(bucket, key, initResult.UploadId);
    foreach (var tag in partETags)
    {
        completeRequest.PartETags.Add(tag);
    }
    ossClient.CompleteMultipartUpload(completeRequest);
    ```

## Abort a multipart upload

Cancel an in-progress multipart upload to clean up uploaded parts:

```csharp title="abortUpload.cs"
using OSS = AlibabaCloud.OSS.V2;

await client.AbortMultipartUploadAsync(new OSS.Models.AbortMultipartUploadRequest()
{
    Bucket = "my-bucket",
    Key = "large-file.zip",
    UploadId = "your-upload-id"
});
```

## List multipart uploads

```csharp title="listUploads.cs"
using OSS = AlibabaCloud.OSS.V2;

var result = await client.ListMultipartUploadsAsync(new OSS.Models.ListMultipartUploadsRequest()
{
    Bucket = "my-bucket"
});

foreach (var upload in result.Uploads ?? [])
{
    Console.WriteLine($"Key: {upload.Key}, Upload ID: {upload.UploadId}");
}
```

<Tip>
  Always abort incomplete multipart uploads you no longer need. Uploaded parts consume storage until the upload is completed or aborted.
</Tip>
