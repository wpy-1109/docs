# Upload Objects

## Simple upload (String / bytes)

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="putObject.cs"
    using System.Text;
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var content = "Hello, OSS!";
    var bodyStream = new MemoryStream(Encoding.UTF8.GetBytes(content));

    var result = await client.PutObjectAsync(new OSS.Models.PutObjectRequest()
    {
        Bucket = "my-bucket",
        Key = "example.txt",
        Body = bodyStream
    });

    Console.WriteLine($"Upload done. Status: {result.StatusCode}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="putObjectV1.cs"
    using System.Text;
    using Aliyun.OSS;

    var content = Encoding.UTF8.GetBytes("Hello, OSS!");
    using var stream = new MemoryStream(content);
    ossClient.PutObject("my-bucket", "example.txt", stream);
    ```
  </Tab>
</Tabs>

## Upload a local file

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="putObjectFromFile.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var result = await client.PutObjectFromFileAsync(new()
    {
        Bucket = "my-bucket",
        Key = "documents/report.pdf"
    }, "/path/to/report.pdf");

    Console.WriteLine($"File uploaded. Request ID: {result.RequestId}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="putObjectFromFileV1.cs"
    using Aliyun.OSS;

    ossClient.PutObject("my-bucket", "documents/report.pdf", "/path/to/report.pdf");
    ```
  </Tab>
</Tabs>

## Upload with metadata

```csharp title="uploadWithMeta.cs"
using System.Text;
using OSS = AlibabaCloud.OSS.V2;

var cfg = OSS.Configuration.LoadDefault();
cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
cfg.Region = "cn-hangzhou";

using var client = new OSS.Client(cfg);

var bodyStream = new MemoryStream(Encoding.UTF8.GetBytes("data"));

var result = await client.PutObjectAsync(new OSS.Models.PutObjectRequest()
{
    Bucket = "my-bucket",
    Key = "data.json",
    Body = bodyStream,
    ContentType = "application/json",
    CacheControl = "max-age=3600"
});

Console.WriteLine($"Uploaded with metadata. Status: {result.StatusCode}");
```

## Append upload (V1)

Append upload allows you to add data to an existing object incrementally:

```csharp title="appendUpload.cs"
using System.Text;
using Aliyun.OSS;

var stream1 = new MemoryStream(Encoding.UTF8.GetBytes("First part. "));
var result = ossClient.AppendObject(new AppendObjectRequest("my-bucket", "append.txt")
{
    Content = stream1,
    Position = 0
});

var stream2 = new MemoryStream(Encoding.UTF8.GetBytes("Second part."));
ossClient.AppendObject(new AppendObjectRequest("my-bucket", "append.txt")
{
    Content = stream2,
    Position = result.NextAppendPosition
});
```

## Upload with progress tracking (V1)

```csharp title="uploadProgress.cs"
using Aliyun.OSS;

using var fileStream = File.OpenRead("/path/to/largefile.zip");
var request = new PutObjectRequest("my-bucket", "largefile.zip", fileStream);

request.StreamTransferProgress += (sender, args) =>
{
    var percent = (double)args.TransferredBytes / args.TotalBytes * 100;
    Console.WriteLine($"Progress: {percent:F1}%");
};

ossClient.PutObject(request);
```
