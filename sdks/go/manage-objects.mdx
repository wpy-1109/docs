# Manage Objects

## List objects

<Tabs>
  <Tab title="V2 SDK">
    ```go title="list_objects_v2.go"
    result, err := client.ListObjectsV2(ctx, &oss.ListObjectsV2Request{
        Bucket: oss.Ptr("my-bucket"),
        Prefix: oss.Ptr("photos/"),
    })
    for _, obj := range result.Contents {
        fmt.Printf("Key: %s, Size: %d\n", oss.ToString(obj.Key), obj.Size)
    }
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```go title="list_objects_v1.go"
    lsRes, err := bucket.ListObjects(oss.Prefix("photos/"))
    for _, object := range lsRes.Objects {
        fmt.Printf("Key: %s, Size: %d\n", object.Key, object.Size)
    }
    ```
  </Tab>
</Tabs>

## Copy objects

```go title="copy_objects.go"
// Within same bucket
_, err := bucket.CopyObject("source-key", "dest-key")

// Cross-bucket
_, err := bucket.CopyObjectFrom("source-bucket", "source-key", "dest-key")
```

## Delete objects

<Tabs>
  <Tab title="V2 SDK">
    ```go title="delete_v2.go"
    _, err := client.DeleteObject(ctx, &oss.DeleteObjectRequest{
        Bucket: oss.Ptr("my-bucket"),
        Key:    oss.Ptr("example.txt"),
    })
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```go title="delete_v1.go"
    // Single delete
    err := bucket.DeleteObject("example.txt")

    // Batch delete
    keys := []string{"key1", "key2", "key3"}
    _, err := bucket.DeleteObjects(keys)
    ```
  </Tab>
</Tabs>

## Object metadata

```go title="metadata.go"
// Upload with metadata
options := []oss.Option{
    oss.ContentType("text/plain"),
    oss.Meta("author", "example-user"),
}
bucket.PutObject("example.txt", strings.NewReader("content"), options...)
```

## Object ACL

```go title="object_acl.go"
// Set object ACL
err := bucket.SetObjectACL("example.txt", oss.ACLPublicRead)

// Get object ACL
result, err := bucket.GetObjectACL("example.txt")
```
