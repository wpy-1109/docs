# Access Control

## STS authentication (Recommended)

Use STS temporary credentials with automatic refresh:

```swift title="stsAuth.swift"
let credential = OSSFederationCredentialProvider { () -> OSSFederationToken? in
    // Call your backend to get a fresh STS token
    let token = OSSFederationToken()
    token.tpiAccessKeyId = "sts-access-key-id"
    token.tpiAccessKeySecret = "sts-access-key-secret"
    token.tpiSecurityToken = "sts-security-token"
    token.expirationTimeInGMTFormat = "2026-01-01T00:00:00Z"
    return token
}

let client = OSSClient(endpoint: endpoint, credentialProvider: credential)
```

## Presigned URLs

Generate temporary access URLs:

```swift title="presignedUrl.swift"
// Generate a signed download URL (valid for 1 hour)
let task = client.presignConstrainURL(
    withBucketName: "my-bucket",
    withObjectKey: "documents/report.pdf",
    withExpirationInterval: 3600
)

task.continue({ t -> Any? in
    if let result = t.result as? NSString {
        print("Download URL: \(result)")
    }
    return nil
})
```

## Error handling

```swift title="errorHandling.swift"
task.continue({ t -> Any? in
    if let error = t.error as NSError? {
        if error.domain == OSSClientErrorDomain {
            print("Client Error: \(error.localizedDescription)")
        } else if error.domain == OSSServerErrorDomain {
            print("Error Code: \(error.userInfo["Code"] ?? "")")
            print("Request ID: \(error.userInfo["RequestId"] ?? "")")
            print("Message: \(error.userInfo["Message"] ?? "")")
        }
    }
    return nil
})
```

<Warning>
  Never embed long-term AccessKey credentials in your iOS app. App binaries can be reverse-engineered, exposing any embedded secrets.
</Warning>
