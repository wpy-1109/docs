# Installation

## Requirements

- iOS 9.0 or later
- Xcode 11 or later

## Install via CocoaPods

Add to your `Podfile`:

```ruby title="Podfile"
pod 'AliyunOSSiOS', '~> 2.10'
```

Then run:

```bash
pod install
```

## Install via carthage

Add to your `Cartfile`:

```text title="Cartfile"
github "aliyun/aliyun-oss-ios-sdk"
```

## Import the SDK

```swift title="Import"
import AliyunOSSiOS
```

```objc title="Import (Objective-C)"
#import <AliyunOSSiOS/AliyunOSSiOS.h>
```

<Warning>
  Mobile devices are untrusted environments. Never embed long-term AccessKey credentials in your app. Use STS temporary credentials instead.
</Warning>
