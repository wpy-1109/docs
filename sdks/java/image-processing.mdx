# Image Processing

Apply real-time image transformations (resize, watermark, format conversion) when downloading objects from OSS.

## Resize an image

```java title="ResizeImage.java"
GetObjectRequest request = new GetObjectRequest(bucketName, "photo.jpg");
request.setProcess("image/resize,m_fixed,w_200,h_200");
OSSObject ossObject = ossClient.getObject(request);

// Save the processed image
InputStream is = ossObject.getObjectContent();
// ... write to file
ossObject.close();
```

## Add a watermark

```java title="WatermarkImage.java"
GetObjectRequest request = new GetObjectRequest(bucketName, "photo.jpg");
request.setProcess("image/watermark,text_SGVsbG8gT1NT,type_d3F5LXplbmhlaQ,size_30");
OSSObject ossObject = ossClient.getObject(request);
```

## Convert image format

```java title="ConvertFormat.java"
GetObjectRequest request = new GetObjectRequest(bucketName, "photo.jpg");
request.setProcess("image/format,png");
OSSObject ossObject = ossClient.getObject(request);
```

## Pipeline multiple operations

Chain multiple operations with the `/` separator:

```java title="PipelineProcess.java"
GetObjectRequest request = new GetObjectRequest(bucketName, "photo.jpg");
request.setProcess("image/resize,w_200/watermark,text_SGVsbG8/format,jpg");
OSSObject ossObject = ossClient.getObject(request);
```

## Use image styles

Apply a predefined image style:

```java title="ImageStyle.java"
GetObjectRequest request = new GetObjectRequest(bucketName, "photo.jpg");
request.setProcess("style/my-style-name");
OSSObject ossObject = ossClient.getObject(request);
```

## Common processing parameters

| Operation | Parameter Format | Description |
|-----------|-----------------|-------------|
| Resize | `image/resize,w_{width},h_{height}` | Resize to specified dimensions |
| Crop | `image/crop,w_{width},h_{height},x_{x},y_{y}` | Crop to specified area |
| Rotate | `image/rotate,{angle}` | Rotate by degrees |
| Watermark | `image/watermark,text_{base64text}` | Add text watermark |
| Format | `image/format,{format}` | Convert to jpg, png, webp, etc. |
| Quality | `image/quality,q_{1-100}` | Set JPEG quality |
