# Installation

The Alibaba Cloud OSS Java SDK is available in two versions. The V1 SDK is production-stable; the V2 SDK is in developer preview with modern builder patterns.

## Prerequisites

- Java 1.8 or later
- Maven or Gradle build tool

## Maven installation (V1)

Add the following dependency to your `pom.xml`:

```xml title="pom.xml"
<dependency>
    <groupId>com.aliyun.oss</groupId>
    <artifactId>aliyun-sdk-oss</artifactId>
    <version>3.18.1</version>
</dependency>
```

For **Java 9+**, add JAXB dependencies:

```xml title="pom.xml (Java 9+)"
<dependency>
    <groupId>javax.xml.bind</groupId>
    <artifactId>jaxb-api</artifactId>
    <version>2.3.1</version>
</dependency>
<dependency>
    <groupId>javax.activation</groupId>
    <artifactId>activation</artifactId>
    <version>1.1.1</version>
</dependency>
<dependency>
    <groupId>org.glassfish.jaxb</groupId>
    <artifactId>jaxb-runtime</artifactId>
    <version>2.3.3</version>
</dependency>
```

## Maven installation (V2 -- preview)

```xml title="pom.xml"
<dependency>
    <groupId>com.aliyun.oss</groupId>
    <artifactId>alibabacloud-oss-sdk-java-v2</artifactId>
    <version>LATEST</version>
</dependency>
```

## Credentials provider dependency

For the credential provider chain (environment variables, instance role, etc.):

```xml title="pom.xml"
<dependency>
    <groupId>com.aliyun</groupId>
    <artifactId>credentials-java</artifactId>
    <version>LATEST</version>
</dependency>
```

## Verify installation

```java title="VerifyInstall.java"
import com.aliyun.oss.OSS;
import com.aliyun.oss.OSSClientBuilder;

public class VerifyInstall {
    public static void main(String[] args) {
        System.out.println("OSS SDK installed successfully!");
    }
}
```
