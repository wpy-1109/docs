# Manage objects

## List objects

```javascript title="listObjects.js"
const result = await client.list({
  prefix: 'photos/',
  'max-keys': 100
});

result.objects.forEach(obj => {
  console.log(`Key: ${obj.name}, Size: ${obj.size}, Modified: ${obj.lastModified}`);
});
```

## List with pagination

```javascript title="listPaginated.js"
let marker = '';
let isTruncated = true;

while (isTruncated) {
  const result = await client.list({
    prefix: 'documents/',
    'max-keys': 50,
    marker
  });

  result.objects?.forEach(obj => console.log(obj.name));
  isTruncated = result.isTruncated;
  marker = result.nextMarker;
}
```

## Copy an object

```javascript title="copyObject.js"
const result = await client.copy('dest-key.txt', 'source-key.txt');
console.log('Copied:', result);
```

## Delete an object

```javascript title="deleteObject.js"
const result = await client.delete('old-file.txt');
console.log('Deleted:', result);
```

## Delete multiple objects

```javascript title="deleteMultiple.js"
const result = await client.deleteMulti(['file1.txt', 'file2.txt', 'file3.txt']);
console.log('Deleted:', result.deleted);
```

## Check if an object exists

```javascript title="objectExists.js"
try {
  await client.head('example.txt');
  console.log('Object exists.');
} catch (e) {
  if (e.code === 'NoSuchKey') {
    console.log('Object does not exist.');
  }
}
```

## Get object metadata

```javascript title="objectMetadata.js"
const result = await client.head('example.txt');
console.log('Content-Type:', result.res.headers['content-type']);
console.log('Content-Length:', result.res.headers['content-length']);
console.log('ETag:', result.res.headers.etag);
```
