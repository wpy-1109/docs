# Upload Objects

## Simple upload

```javascript title="simpleUpload.js"
// Upload a File object from an <input type="file"> element
const file = document.getElementById('fileInput').files[0];
const result = await client.put(`uploads/${file.name}`, file);
console.log('Uploaded:', result.url);
```

## Upload with custom headers

```javascript title="uploadWithHeaders.js"
const result = await client.put('data.json', file, {
  headers: {
    'Content-Type': 'application/json',
    'Cache-Control': 'max-age=3600',
    'x-oss-object-acl': 'private'
  }
});
```

## Multipart upload

For large files, use multipart upload:

```javascript title="multipartUpload.js"
const result = await client.multipartUpload('videos/large-video.mp4', file, {
  partSize: 1 * 1024 * 1024, // 1 MB per part
  progress: (p, checkpoint) => {
    console.log(`Progress: ${(p * 100).toFixed(1)}%`);
    // Save checkpoint for resumable upload
    localStorage.setItem('uploadCheckpoint', JSON.stringify(checkpoint));
  }
});

console.log('Multipart upload complete:', result);
```

## Resumable upload

Resume an interrupted upload using a saved checkpoint:

```javascript title="resumableUpload.js"
let checkpoint = JSON.parse(localStorage.getItem('uploadCheckpoint'));

const result = await client.multipartUpload('videos/large-video.mp4', file, {
  checkpoint,
  progress: (p, cp) => {
    checkpoint = cp;
    localStorage.setItem('uploadCheckpoint', JSON.stringify(cp));
    console.log(`Progress: ${(p * 100).toFixed(1)}%`);
  }
});

localStorage.removeItem('uploadCheckpoint');
```

## Cancel an upload

```javascript title="cancelUpload.js"
client.cancel();
```
