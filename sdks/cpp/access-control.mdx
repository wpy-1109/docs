# Access Control

## Bucket ACL

### Set bucket ACL

```cpp title="setBucketAcl.cpp"
#include <alibabacloud/oss/OssClient.h>
using namespace AlibabaCloud::OSS;

// ... (initialization) ...

auto outcome = client.SetBucketAcl("my-bucket", CannedAccessControlList::Private);

if (outcome.isSuccess()) {
    std::cout << "Bucket ACL set to private." << std::endl;
}
```

Available ACL values:

| ACL | Constant | Description |
|-----|----------|-------------|
| Private | `CannedAccessControlList::Private` | Only the owner can read/write |
| Public Read | `CannedAccessControlList::PublicRead` | Anyone can read; only owner can write |
| Public Read-Write | `CannedAccessControlList::PublicReadWrite` | Anyone can read and write |

### Get bucket ACL

```cpp title="getBucketAcl.cpp"
auto outcome = client.GetBucketAcl("my-bucket");

if (outcome.isSuccess()) {
    std::cout << "ACL: " << outcome.result().Acl() << std::endl;
}
```

## Object ACL

### Set object ACL

```cpp title="setObjectAcl.cpp"
SetObjectAclRequest request("my-bucket", "example.txt");
request.setAcl(CannedAccessControlList::PublicRead);

auto outcome = client.SetObjectAcl(request);
```

### Get object ACL

```cpp title="getObjectAcl.cpp"
auto outcome = client.GetObjectAcl(GetObjectAclRequest("my-bucket", "example.txt"));

if (outcome.isSuccess()) {
    std::cout << "Object ACL: " << outcome.result().Acl() << std::endl;
}
```

## Presigned URLs

Generate temporary access URLs for sharing private objects:

```cpp title="presignedUrl.cpp"
// Generate a download URL (valid for 1 hour)
auto url = client.GeneratePresignedUrl("my-bucket", "documents/report.pdf", 3600, Http::Get);
std::cout << "Download URL: " << url << std::endl;

// Generate an upload URL (valid for 1 hour)
auto uploadUrl = client.GeneratePresignedUrl("my-bucket", "uploads/file.txt", 3600, Http::Put);
std::cout << "Upload URL: " << uploadUrl << std::endl;
```

## CORS configuration

```cpp title="cors.cpp"
SetBucketCorsRequest request("my-bucket");

CORSRule rule;
rule.addAllowedOrigin("https://example.com");
rule.addAllowedMethod("GET");
rule.addAllowedMethod("PUT");
rule.addAllowedHeader("*");
rule.setMaxAgeSeconds(3600);

request.addCORSRule(rule);
client.SetBucketCors(request);
```

## Bucket policy

```cpp title="bucketPolicy.cpp"
std::string policy = R"({
    "Version": "1",
    "Statement": [{
        "Effect": "Allow",
        "Action": ["oss:GetObject"],
        "Principal": ["*"],
        "Resource": ["acs:oss:*:*:my-bucket/public/*"]
    }]
})";

auto outcome = client.SetBucketPolicy("my-bucket", policy);
```

<Warning>
  Use `PublicRead` and `PublicReadWrite` ACLs with caution. For most use cases, keep ACL as `Private` and use presigned URLs or RAM policies for access control.
</Warning>
