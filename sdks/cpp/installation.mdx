# Installation

## Requirements

- C++11 or later
- Visual Studio 2013+ (Windows)
- GCC 4.8+ (Linux)
- Clang 3.3+ (macOS)
- CMake 3.1+

## Download the SDK

- [Download ZIP](https://github.com/aliyun/aliyun-oss-cpp-sdk/archive/refs/tags/1.10.0.zip)
- [Clone from GitHub](https://github.com/aliyun/aliyun-oss-cpp-sdk)

## Build from source

<Tabs>
  <Tab title="Linux">
    ```bash title="Build on Linux"
    git clone https://github.com/aliyun/aliyun-oss-cpp-sdk.git
    cd aliyun-oss-cpp-sdk
    mkdir build && cd build
    cmake ..
    make
    sudo make install
    ```
  </Tab>
  <Tab title="macOS">
    ```bash title="Build on macOS"
    brew install cmake openssl curl
    git clone https://github.com/aliyun/aliyun-oss-cpp-sdk.git
    cd aliyun-oss-cpp-sdk
    mkdir build && cd build
    cmake ..
    make
    sudo make install
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell title="Build on Windows"
    git clone https://github.com/aliyun/aliyun-oss-cpp-sdk.git
    cd aliyun-oss-cpp-sdk
    mkdir build && cd build
    cmake .. -G "Visual Studio 16 2019" -A x64
    cmake --build . --config Release
    cmake --install .
    ```
  </Tab>
</Tabs>

## Set up credentials

```bash title="macOS / Linux"
export OSS_ACCESS_KEY_ID=<your-access-key-id>
export OSS_ACCESS_KEY_SECRET=<your-access-key-secret>
```

```powershell title="Windows"
set OSS_ACCESS_KEY_ID=<your-access-key-id>
set OSS_ACCESS_KEY_SECRET=<your-access-key-secret>
```

## Verify installation

```cpp title="verify.cpp"
#include <alibabacloud/oss/OssClient.h>
using namespace AlibabaCloud::OSS;

int main() {
    InitializeSdk();

    ClientConfiguration conf;
    conf.signatureVersion = SignatureVersionType::V4;
    auto credentialsProvider = std::make_shared<EnvironmentVariableCredentialsProvider>();
    OssClient client("https://oss-cn-hangzhou.aliyuncs.com", credentialsProvider, conf);
    client.SetRegion("cn-hangzhou");

    std::cout << "OSS C++ SDK initialized successfully." << std::endl;

    ShutdownSdk();
    return 0;
}
```
