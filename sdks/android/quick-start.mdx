# Quick start

## Upload a file

```java title="uploadFile.java"
PutObjectRequest put = new PutObjectRequest(
    "my-bucket", "uploads/photo.jpg", "/sdcard/DCIM/photo.jpg"
);

// Optional: set progress callback
put.setProgressCallback((request, currentSize, totalSize) -> {
    double progress = (double) currentSize / totalSize * 100;
    Log.d("OSS", "Progress: " + String.format("%.1f%%", progress));
});

// Async upload
OSSAsyncTask task = oss.asyncPutObject(put, new OSSCompletedCallback<PutObjectRequest, PutObjectResult>() {
    @Override
    public void onSuccess(PutObjectRequest request, PutObjectResult result) {
        Log.d("OSS", "Upload success. ETag: " + result.getETag());
    }

    @Override
    public void onFailure(PutObjectRequest request, ClientException clientException, ServiceException serviceException) {
        if (clientException != null) {
            Log.e("OSS", "Client error: " + clientException.getMessage());
        }
        if (serviceException != null) {
            Log.e("OSS", "Service error: " + serviceException.getErrorCode());
        }
    }
});
```

## Download a file

```java title="downloadFile.java"
GetObjectRequest get = new GetObjectRequest("my-bucket", "uploads/photo.jpg");

oss.asyncGetObject(get, new OSSCompletedCallback<GetObjectRequest, GetObjectResult>() {
    @Override
    public void onSuccess(GetObjectRequest request, GetObjectResult result) {
        InputStream inputStream = result.getObjectContent();
        // Read the stream...
        Log.d("OSS", "Download success. Content-Length: " + result.getContentLength());
    }

    @Override
    public void onFailure(GetObjectRequest request, ClientException clientException, ServiceException serviceException) {
        Log.e("OSS", "Download failed");
    }
});
```

## List objects

```java title="listObjects.java"
ListObjectsRequest listRequest = new ListObjectsRequest("my-bucket");
listRequest.setPrefix("photos/");
listRequest.setMaxKeys(100);

oss.asyncListObjects(listRequest, new OSSCompletedCallback<ListObjectsRequest, ListObjectsResult>() {
    @Override
    public void onSuccess(ListObjectsRequest request, ListObjectsResult result) {
        for (OSSObjectSummary obj : result.getObjectSummaries()) {
            Log.d("OSS", "Key: " + obj.getKey() + ", Size: " + obj.getSize());
        }
    }

    @Override
    public void onFailure(ListObjectsRequest request, ClientException clientException, ServiceException serviceException) {
        Log.e("OSS", "List failed");
    }
});
```

## Delete an object

```java title="deleteObject.java"
DeleteObjectRequest delete = new DeleteObjectRequest("my-bucket", "old-file.txt");
oss.asyncDeleteObject(delete, new OSSCompletedCallback<DeleteObjectRequest, DeleteObjectResult>() {
    @Override
    public void onSuccess(DeleteObjectRequest request, DeleteObjectResult result) {
        Log.d("OSS", "Deleted.");
    }

    @Override
    public void onFailure(DeleteObjectRequest request, ClientException clientException, ServiceException serviceException) {
        Log.e("OSS", "Delete failed");
    }
});
```
