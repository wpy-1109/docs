# Manage objects

## List objects

```java title="listObjects.java"
ListObjectsRequest request = new ListObjectsRequest("my-bucket");
request.setPrefix("photos/");
request.setMaxKeys(100);

oss.asyncListObjects(request, new OSSCompletedCallback<ListObjectsRequest, ListObjectsResult>() {
    @Override
    public void onSuccess(ListObjectsRequest request, ListObjectsResult result) {
        for (OSSObjectSummary obj : result.getObjectSummaries()) {
            Log.d("OSS", "Key: " + obj.getKey() + ", Size: " + obj.getSize());
        }
    }
    @Override
    public void onFailure(ListObjectsRequest request, ClientException ce, ServiceException se) {
        Log.e("OSS", "List failed");
    }
});
```

## Check if an object exists

```java title="objectExists.java"
try {
    oss.doesObjectExist("my-bucket", "example.txt");
    Log.d("OSS", "Object exists.");
} catch (Exception e) {
    Log.d("OSS", "Object does not exist.");
}
```

## Copy an object

```java title="copyObject.java"
CopyObjectRequest copy = new CopyObjectRequest(
    "source-bucket", "source-key.txt",
    "dest-bucket", "dest-key.txt"
);

oss.asyncCopyObject(copy, new OSSCompletedCallback<CopyObjectRequest, CopyObjectResult>() {
    @Override
    public void onSuccess(CopyObjectRequest request, CopyObjectResult result) {
        Log.d("OSS", "Copied.");
    }
    @Override
    public void onFailure(CopyObjectRequest request, ClientException ce, ServiceException se) {
        Log.e("OSS", "Copy failed");
    }
});
```

```java title="deleteObject.java"
    @Override
    public void onFailure(DeleteObjectRequest request, ClientException ce, ServiceException se) {

```

## Get object metadata

```java title="objectMetadata.java"
HeadObjectRequest head = new HeadObjectRequest("my-bucket", "example.txt");

oss.asyncHeadObject(head, new OSSCompletedCallback<HeadObjectRequest, HeadObjectResult>() {
    @Override
    public void onSuccess(HeadObjectRequest request, HeadObjectResult result) {
        Log.d("OSS", "Content-Type: " + result.getMetadata().getContentType());
        Log.d("OSS", "Content-Length: " + result.getMetadata().getContentLength());
    }
    @Override
    public void onFailure(HeadObjectRequest request, ClientException ce, ServiceException se) {
        Log.e("OSS", "Head failed");
    }
});
```
