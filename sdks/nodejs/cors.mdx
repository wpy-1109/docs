# CORS

Configure CORS rules to allow browser-based access to your OSS resources.

## Set CORS rules

```javascript title="setCors.js"
const rules = [
  {
    allowedOrigin: ['https://example.com', 'https://app.example.com'],
    allowedMethod: ['GET', 'PUT', 'POST', 'DELETE'],
    allowedHeader: ['*'],
    exposeHeader: ['ETag', 'x-oss-request-id'],
    maxAgeSeconds: 3600,
  },
];

await client.putBucketCORS('my-bucket', rules);
```

## Get CORS rules

```javascript title="getCors.js"
const result = await client.getBucketCORS('my-bucket');
result.rules.forEach(rule => {
  console.log('Origins:', rule.allowedOrigin);
  console.log('Methods:', rule.allowedMethod);
});
```

## Delete CORS rules

```javascript title="deleteCors.js"
await client.deleteBucketCORS('my-bucket');
```
