# Installation

The Alibaba Cloud OSS Node.js SDK (`ali-oss`) works in both Node.js server environments and browser environments.

## Prerequisites

- Node.js 12.x or later
- npm or yarn package manager

## Install via npm

```bash
npm install ali-oss
```

## Install via yarn

```bash
yarn add ali-oss
```

## Verify installation

```javascript title="verify.js"
const OSS = require('ali-oss');
console.log('OSS SDK for Node.js installed successfully');
```
