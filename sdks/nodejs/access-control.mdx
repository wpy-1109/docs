# Access Control

## Bucket ACL

```javascript title="bucketAcl.js"
// Set bucket ACL
await client.putBucketACL('my-bucket', 'private');
// Options: 'private', 'public-read', 'public-read-write'

// Get bucket ACL
const result = await client.getBucketACL('my-bucket');
console.log('ACL:', result.acl);
```

## Object ACL

```javascript title="objectAcl.js"
// Set object ACL
await client.putACL('example.txt', 'public-read');

// Get object ACL
const acl = await client.getACL('example.txt');
console.log('ACL:', acl.acl);
```

## Bucket policy

```javascript title="bucketPolicy.js"
const policy = {
  Version: '1',
  Statement: [{
    Effect: 'Allow',
    Principal: ['*'],
    Action: ['oss:GetObject'],
    Resource: ['acs:oss:*:*:my-bucket/public/*'],
  }],
};

await client.putBucketPolicy('my-bucket', JSON.stringify(policy));

// Get policy
const result = await client.getBucketPolicy('my-bucket');
console.log('Policy:', result.policy);

// Delete policy
await client.deleteBucketPolicy('my-bucket');
```

## STS temporary credentials

```javascript title="stsAuth.js"
const client = new OSS({
  region: 'oss-cn-hangzhou',
  accessKeyId: 'temporary-access-key-id',
  accessKeySecret: 'temporary-access-key-secret',
  stsToken: 'security-token',
  bucket: 'my-bucket',
  authorizationV4: true,
});
```
