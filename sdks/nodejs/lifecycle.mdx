# Lifecycle rules

## Set lifecycle rules

```javascript title="setLifecycle.js"
const rules = [
  {
    id: 'expire-logs',
    prefix: 'logs/',
    status: 'Enabled',
    expiration: { days: 30 },
  },
  {
    id: 'transition-to-ia',
    prefix: 'data/',
    status: 'Enabled',
    transition: { days: 60, storageClass: 'IA' },
  },
];

await client.putBucketLifecycle('my-bucket', rules);
```

## Get lifecycle rules

```javascript title="getLifecycle.js"
const result = await client.getBucketLifecycle('my-bucket');
result.rules.forEach(rule => {
  console.log('Rule:', rule.id, 'Prefix:', rule.prefix, 'Status:', rule.status);
});
```

## Delete lifecycle rules

```javascript title="deleteLifecycle.js"
await client.deleteBucketLifecycle('my-bucket');
```
