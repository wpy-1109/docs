# Image processing

## Resize an image

```javascript title="resizeImage.js"
const result = await client.get('photo.jpg', {
  process: 'image/resize,w_200,h_200',
});
// result.content contains the resized image data
```

## Apply multiple operations

```javascript title="pipelineProcess.js"
const result = await client.get('photo.jpg', {
  process: 'image/resize,w_400/watermark,text_SGVsbG8/format,png',
});
```

## Use image styles

```javascript title="imageStyle.js"
const result = await client.get('photo.jpg', {
  process: 'style/my-thumbnail-style',
});
```

## Generate a processed image URL

```javascript title="processedUrl.js"
const url = client.signatureUrl('photo.jpg', {
  expires: 3600,
  process: 'image/resize,w_200,h_200/format,webp',
});
console.log('Processed image URL:', url);
```
