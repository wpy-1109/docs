# Static Website Hosting

## Enable static website hosting

```javascript title="enableWebsite.js"
await client.putBucketWebsite('my-bucket', {
  index: 'index.html',
  error: '404.html',
});
```

## Get website configuration

```javascript title="getWebsite.js"
const result = await client.getBucketWebsite('my-bucket');
console.log('Index:', result.index);
console.log('Error:', result.error);
```

## Delete website configuration

```javascript title="deleteWebsite.js"
await client.deleteBucketWebsite('my-bucket');
```
