# Versioning

## Enable versioning

```javascript title="enableVersioning.js"
await client.putBucketVersioning('my-bucket', 'Enabled');
```

## Suspend versioning

```javascript title="suspendVersioning.js"
await client.putBucketVersioning('my-bucket', 'Suspended');
```

## Get versioning status

```javascript title="getVersioning.js"
const result = await client.getBucketVersioning('my-bucket');
console.log('Status:', result.status);
```

## Download a specific version

```javascript title="getVersion.js"
const result = await client.get('example.txt', {
  versionId: 'version-id-here',
});
```

## List object versions

```javascript title="listVersions.js"
const result = await client.getBucketVersions({
  'max-keys': 100,
});
result.objects.forEach(obj => {
  console.log('Key:', obj.name, 'VersionId:', obj.versionId,
    'IsLatest:', obj.isLatest);
});
```

## Delete a specific version

```javascript title="deleteVersion.js"
await client.delete('example.txt', { versionId: 'version-id-here' });
```
