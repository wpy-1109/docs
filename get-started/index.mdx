# Alibaba Cloud OSS Documentation

Alibaba Cloud Object Storage Service (OSS) is a secure, cost-effective, and highly reliable cloud storage service with 99.9999999999% (twelve 9s) data durability and 99.995% availability. Store, manage, and access any amount of data from anywhere.
