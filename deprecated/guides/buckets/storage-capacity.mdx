You can monitor the storage capacity and resource usage of your buckets to track costs and optimize storage. The OSS console provides charts for storage usage, bandwidth, traffic, and API request counts.

## Prerequisites

- SDK installed and configured ([quickstart](/quickstart))
- An Alibaba Cloud account with OSS activated
- Access to the OSS console or API credentials

## Available metrics

| Category | Metrics |
|----------|---------|
| **Storage** | Total storage, storage by class (Standard, IA, Archive), billed storage |
| **Bandwidth** | Public inbound/outbound, internal inbound/outbound, CDN outbound |
| **Traffic** | Cumulative traffic by type |
| **Requests** | GET, PUT, and other API request counts |
| **Image Processing** | Image processing capacity and request count |

<Note>
Ranking Statistics, Region and Operator Statistics, API Statistics, and Object Access Statistics are available only in selected regions: Chinese mainland, China (Hong Kong), US (Virginia), US (Silicon Valley), Germany (Frankfurt), and Singapore.
</Note>

## Using the console

<Steps>
  <Step title="Navigate to your bucket">
    Log on to the [OSS console](https://oss.console.alibabacloud.com/) and click the bucket name.
  </Step>
  <Step title="View usage data">
    In the left navigation pane, choose **Data Usage > Basic Data**. The page displays charts for storage, bandwidth, traffic, requests, and image processing.
  </Step>
  <Step title="Adjust the time range">
    Use the time picker at the top to select the period you want to analyze.
  </Step>
</Steps>

## Using an SDK

You can query bucket statistics programmatically using the `GetBucketStat` API:

<Tabs>
  <Tab title="Python">
    ```python title="storage_usage.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    result = client.get_bucket_stat(oss.GetBucketStatRequest(
        bucket="examplebucket"
    ))

    print(f"Storage: {result.storage} bytes")
    print(f"Object Count: {result.object_count}")
    print(f"Multipart Upload Count: {result.multipart_upload_count}")
    ```
  </Tab>
  <Tab title="Java">
    ```java title="StorageUsage.java"
    import com.aliyun.oss.*;
    import com.aliyun.oss.common.auth.*;
    import com.aliyun.oss.common.comm.SignVersion;
    import com.aliyun.oss.model.BucketStat;

    public class StorageUsage {
        public static void main(String[] args) throws Exception {
            String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
            String region = "cn-hangzhou";
            EnvironmentVariableCredentialsProvider credentialsProvider =
                CredentialsProviderFactory.newEnvironmentVariableCredentialsProvider();

            ClientBuilderConfiguration clientConfig = new ClientBuilderConfiguration();
            clientConfig.setSignatureVersion(SignVersion.V4);
            OSS ossClient = OSSClientBuilder.create()
                .endpoint(endpoint)
                .credentialsProvider(credentialsProvider)
                .clientConfiguration(clientConfig)
                .region(region)
                .build();

            try {
                BucketStat stat = ossClient.getBucketStat("examplebucket");
                System.out.println("Storage: " + stat.getStorageSize() + " bytes");
                System.out.println("Object Count: " + stat.getObjectCount());
            } finally {
                ossClient.shutdown();
            }
        }
    }
    ```
  </Tab>
</Tabs>
