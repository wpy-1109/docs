Pay-by-requester shifts data-access costs (traffic and request fees) from the bucket owner to the requester. The bucket owner pays only for storage. When enabled, anonymous access is disabled and all requests must include authentication.

## How it works

OSS processes each request as follows:

1. If the request includes the `x-oss-request-payer: requester` header, OSS authenticates the requester and bills them for traffic and request fees.
2. If the header is absent and the requester is the bucket owner, OSS processes the request normally.
3. If the header is absent and the requester is not the owner, OSS rejects the request.

<Warning>
When pay-by-requester is enabled, anonymous access is no longer possible. All requests must include valid authentication credentials.
</Warning>

## Prerequisites

- SDK installed and configured ([quickstart](/quickstart))
- An Alibaba Cloud account with OSS activated
- An AccessKey pair or STS temporary credentials
- Bucket owner permissions (`oss:PutBucketRequestPayment`)

## Enable Pay-by-Requester

<Tabs>
  <Tab title="Console">
    <Steps>
      <Step title="Navigate to settings">
        Log on to the [OSS console](https://oss.console.alibabacloud.com/), click the bucket name, then choose **Bucket Settings > Pay-by-requester**.
      </Step>
      <Step title="Enable">
        Turn on the **Pay-by-requester** switch and click **OK** in the confirmation dialog.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Python">
    ```python title="pay_by_requester.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    result = client.put_bucket_request_payment(oss.PutBucketRequestPaymentRequest(
        bucket="examplebucket",
        request_payment_configuration=oss.RequestPaymentConfiguration(
            payer="Requester"
        )
    ))

    print(f"Pay-by-requester enabled. Status: {result.status_code}")
    ```
  </Tab>
  <Tab title="Java">
    ```java title="PayByRequester.java"
    import com.aliyun.oss.*;
    import com.aliyun.oss.common.auth.*;
    import com.aliyun.oss.common.comm.SignVersion;
    import com.aliyun.oss.model.Payer;

    public class PayByRequester {
        public static void main(String[] args) throws Exception {
            String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
            String region = "cn-hangzhou";
            EnvironmentVariableCredentialsProvider credentialsProvider =
                CredentialsProviderFactory.newEnvironmentVariableCredentialsProvider();

            ClientBuilderConfiguration clientConfig = new ClientBuilderConfiguration();
            clientConfig.setSignatureVersion(SignVersion.V4);
            OSS ossClient = OSSClientBuilder.create()
                .endpoint(endpoint)
                .credentialsProvider(credentialsProvider)
                .clientConfiguration(clientConfig)
                .region(region)
                .build();

            try {
                ossClient.setBucketRequestPayment("examplebucket", Payer.Requester);
                System.out.println("Pay-by-requester enabled.");
            } finally {
                ossClient.shutdown();
            }
        }
    }
    ```
  </Tab>
</Tabs>

## Access a Pay-by-Requester bucket

When accessing a bucket with pay-by-requester enabled, include the `x-oss-request-payer: requester` header:

```bash
# ossutil example
ossutil cp oss://examplebucket/example.txt ./example.txt --payer requester
```
