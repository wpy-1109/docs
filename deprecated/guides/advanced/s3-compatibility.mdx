Alibaba Cloud OSS provides an S3-compatible API layer that allows you to use AWS S3 SDKs, CLI tools, and third-party applications with your OSS buckets.

## Supported operations

OSS supports the most commonly used S3 API operations:

| Category | Supported Operations |
|----------|---------------------|
| Bucket | CreateBucket, DeleteBucket, ListBuckets, HeadBucket |
| Object | GetObject, PutObject, DeleteObject, CopyObject, HeadObject, ListObjects, ListObjectsV2 |
| Multipart | CreateMultipartUpload, UploadPart, CompleteMultipartUpload, AbortMultipartUpload, ListParts |
| ACL | GetBucketAcl, PutBucketAcl, GetObjectAcl, PutObjectAcl |

## Configure the AWS CLI

```bash
aws configure
# AWS Access Key ID: <your-oss-access-key-id>
# AWS Secret Access Key: <your-oss-access-key-secret>
# Default region name: oss-cn-hangzhou
# Default output format: json
```

Use the `--endpoint-url` flag to point to your OSS endpoint:

```bash
aws s3 ls s3://examplebucket --endpoint-url https://oss-cn-hangzhou.aliyuncs.com
```

## Prerequisites

- SDK installed and configured ([quickstart](/quickstart))

## Configure the AWS SDK (Python boto3)

```python
import boto3

s3 = boto3.client(
    "s3",
    endpoint_url="https://oss-cn-hangzhou.aliyuncs.com",
    aws_access_key_id="<your-oss-access-key-id>",
    aws_secret_access_key="<your-oss-access-key-secret>",
    region_name="oss-cn-hangzhou",
)

# List objects
response = s3.list_objects_v2(Bucket="examplebucket")
for obj in response.get("Contents", []):
    print(obj["Key"])
```

## Limitations

- Not all S3 features are supported. Features like S3 Select, S3 Object Lambda, and S3 Glacier Flexible Retrieval are not available through the S3 compatibility layer.
- Use path-style addressing (`https://endpoint/bucket/key`) if virtual-hosted style does not work with your tool.
- Some tools may require additional configuration for V4 signature compatibility.

<Tip>
For full feature coverage and best performance, use the native [OSS SDKs](/sdks/overview) instead of S3-compatible tools.
</Tip>
