# Upload objects

## Upload from a string/Reader

<Tabs>
  <Tab title="V2 SDK">
    ```go title="upload_string_v2.go"
    result, err := client.PutObject(ctx, &oss.PutObjectRequest{
        Bucket:      oss.Ptr("my-bucket"),
        Key:         oss.Ptr("documents/hello.txt"),
        Body:        strings.NewReader("Hello, Alibaba Cloud OSS!"),
        ContentType: oss.Ptr("text/plain"),
    })
    fmt.Printf("ETag: %s\n", oss.ToString(result.ETag))
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```go title="upload_string_v1.go"
    bucket, _ := client.Bucket("my-bucket")
    err := bucket.PutObject("hello.txt", strings.NewReader("Hello OSS!"))
    ```
  </Tab>
</Tabs>

## Upload a local file

<Tabs>
  <Tab title="V2 SDK">
    ```go title="upload_file_v2.go"
    file, err := os.Open("/path/to/local/file.jpg")
    if err != nil {
        log.Fatal(err)
    }
    defer file.Close()
    fileStat, _ := file.Stat()

    result, err := client.PutObject(ctx, &oss.PutObjectRequest{
        Bucket:        oss.Ptr("my-bucket"),
        Key:           oss.Ptr("images/photo.jpg"),
        Body:          file,
        ContentType:   oss.Ptr("image/jpeg"),
        ContentLength: oss.Ptr(fileStat.Size()),
    })
    fmt.Printf("ETag: %s\n", oss.ToString(result.ETag))
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```go title="upload_file_v1.go"
    bucket, _ := client.Bucket("my-bucket")
    err := bucket.PutObjectFromFile("images/photo.jpg", "/path/to/photo.jpg")
    ```
  </Tab>
</Tabs>

## Upload with metadata

```go title="upload_metadata.go"
options := []oss.Option{
    oss.ContentType("text/plain"),
    oss.Meta("author", "example-user"),
    oss.ObjectACL(oss.ACLPublicRead),
}
err := bucket.PutObject("example.txt", strings.NewReader("content"), options...)
```
