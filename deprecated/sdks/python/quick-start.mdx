# Quick start

<Info>
This quickstart takes approximately 5 minutes to complete.
</Info>

## Prerequisites

- Python 3.x installed
- An Alibaba Cloud account with OSS activated
- AccessKey ID and Secret configured as environment variables

## V2 SDK quick start

<Steps>
  <Step title="Install the SDK">
    ```bash
    pip install alibabacloud-oss-v2
    ```
  </Step>

  <Step title="Set Environment Variables">
    ```bash
    export OSS_ACCESS_KEY_ID="your-access-key-id"
    export OSS_ACCESS_KEY_SECRET="your-access-key-secret"
    ```
  </Step>

  <Step title="Upload, Download, and Manage Objects">
    ```python title="quickstart_v2.py"
    import alibabacloud_oss_v2 as oss

    def main():
        region = "cn-hangzhou"
        bucket_name = "your-bucket-name"
        object_name = "quickstart/hello.txt"

        # Initialize client
        credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
        cfg = oss.config.load_default()
        cfg.credentials_provider = credentials_provider
        cfg.region = region
        client = oss.Client(cfg)

        # Upload a string
        result = client.put_object(oss.PutObjectRequest(
            bucket=bucket_name,
            key=object_name,
            body=b'Hello, Alibaba Cloud OSS!',
        ))
        print(f'Upload successful, ETag: {result.etag}')

        # Upload a local file
        with open('local-file.txt', 'rb') as f:
            result = client.put_object(oss.PutObjectRequest(
                bucket=bucket_name,
                key='quickstart/file.txt',
                body=f,
            ))
        print(f'File uploaded, ETag: {result.etag}')

        # Download the object
        result = client.get_object(oss.GetObjectRequest(
            bucket=bucket_name,
            key=object_name,
        ))
        content = result.body.read()
        print(f'Downloaded content: {content.decode()}')

    if __name__ == "__main__":
        main()
    ```
  </Step>
</Steps>

## V1 SDK quick start

```python title="quickstart_v1.py"
import oss2
from oss2.credentials import EnvironmentVariableCredentialsProvider

# Initialize
auth = oss2.ProviderAuth(EnvironmentVariableCredentialsProvider())
bucket = oss2.Bucket(auth, 'https://oss-cn-hangzhou.aliyuncs.com', 'your-bucket-name')

# Upload an object
bucket.put_object('quickstart/hello.txt', 'Hello OSS!')
print('Object uploaded')

# Download an object
result = bucket.get_object('quickstart/hello.txt')
print(f'Content: {result.read().decode()}')

# List objects
for obj in oss2.ObjectIterator(bucket):
    print(f'Object: {obj.key}')

# Delete an object
bucket.delete_object('quickstart/hello.txt')
print('Object deleted')
```
