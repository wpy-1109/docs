# Access control

## Bucket ACL

```php title="bucketAcl.php"
$ossClient->putBucketAcl($bucket, OssClient::OSS_ACL_TYPE_PRIVATE);

$acl = $ossClient->getBucketAcl($bucket);
echo "ACL: " . $acl . "\n";
```

## Object ACL

```php title="objectAcl.php"
$ossClient->putObjectAcl($bucket, 'example.txt', OssClient::OSS_ACL_TYPE_PUBLIC_READ);

$acl = $ossClient->getObjectAcl($bucket, 'example.txt');
echo "ACL: " . $acl . "\n";
```

## CORS

```php title="cors.php"
$corsConfig = new \OSS\Model\CorsConfig();
$rule = new \OSS\Model\CorsRule();
$rule->addAllowedOrigin('https://example.com');
$rule->addAllowedMethod('GET');
$rule->addAllowedMethod('PUT');
$rule->addAllowedHeader('*');
$rule->setMaxAgeSeconds(3600);
$corsConfig->addRule($rule);

$ossClient->putBucketCors($bucket, $corsConfig);
```
