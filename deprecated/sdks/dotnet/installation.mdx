# Installation

## Requirements

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    - .NET Framework 4.7.1 or later
    - .NET Standard 2.0 or later
    - .NET 5.0 or later
  </Tab>
  <Tab title="V1 SDK">
    - **Windows:** .NET Framework 2.0 or later, Visual Studio 2010 or later
    - **Linux/Mac:** Mono 3.12 or later
  </Tab>
</Tabs>

## Install via NuGet

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    Install the `AlibabaCloud.OSS.V2` package:

    ```bash title="Package Manager Console"
    Install-Package AlibabaCloud.OSS.V2
    ```

    Or using the .NET CLI:

    ```bash title=".NET CLI"
    dotnet add package AlibabaCloud.OSS.V2
    ```

    <Note>
      V2 SDK package: [AlibabaCloud.OSS.V2 on NuGet](https://www.nuget.org/packages/AlibabaCloud.OSS.V2)
      Source code: [GitHub](https://github.com/aliyun/alibabacloud-oss-csharp-sdk-v2)
    </Note>
  </Tab>
  <Tab title="V1 SDK">
    Install `Aliyun.OSS.SDK` (for .NET Framework) or `Aliyun.OSS.SDK.NetCore` (for .NET Core):

    ```bash title="Package Manager Console (.NET Framework)"
    Install-Package Aliyun.OSS.SDK
    ```

    ```bash title="Package Manager Console (.NET Core)"
    Install-Package Aliyun.OSS.SDK.NetCore
    ```

    <Note>
      Source code: [GitHub](https://github.com/aliyun/aliyun-oss-csharp-sdk)
      [Release history](https://github.com/aliyun/aliyun-oss-csharp-sdk/releases)
    </Note>
  </Tab>
</Tabs>

## Set up credentials

Set environment variables for your AccessKey:

```bash title="macOS / Linux"
export OSS_ACCESS_KEY_ID=<your-access-key-id>
export OSS_ACCESS_KEY_SECRET=<your-access-key-secret>
```

```powershell title="Windows (PowerShell)"
$env:OSS_ACCESS_KEY_ID="<your-access-key-id>"
$env:OSS_ACCESS_KEY_SECRET="<your-access-key-secret>"
```

<Warning>
  Never hard-code credentials in production code. Use environment variables or a credentials provider.
</Warning>

## Verify installation

```csharp title="Program.cs"
using OSS = AlibabaCloud.OSS.V2;

var cfg = OSS.Configuration.LoadDefault();
cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
cfg.Region = "cn-hangzhou";

using var client = new OSS.Client(cfg);
Console.WriteLine("OSS .NET SDK connected successfully.");
```
