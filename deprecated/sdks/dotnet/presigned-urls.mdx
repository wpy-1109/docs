# Presigned URLs

Presigned URLs allow temporary access to private objects without requiring credentials from the end user.

## Generate a download URL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="presignedDownload.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var result = client.Presign(new OSS.Models.GetObjectRequest()
    {
        Bucket = "my-bucket",
        Key = "documents/report.pdf"
    });

    Console.WriteLine($"Download URL: {result.Url}");
    ```

    With custom expiration:

    ```csharp title="presignedDownloadExpiry.cs"
    var result = client.Presign(
        new OSS.Models.GetObjectRequest()
        {
            Bucket = "my-bucket",
            Key = "documents/report.pdf"
        },
        TimeSpan.FromHours(2) // Expires in 2 hours
    );
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="presignedDownloadV1.cs"
    using Aliyun.OSS;

    var request = new GeneratePresignedUriRequest("my-bucket", "documents/report.pdf", SignHttpMethod.Get)
    {
        Expiration = DateTime.UtcNow.AddHours(1)
    };

    var url = ossClient.GeneratePresignedUri(request);
    Console.WriteLine($"Download URL: {url}");
    ```
  </Tab>
</Tabs>

## Generate an upload URL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="presignedUpload.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var result = client.Presign(new OSS.Models.PutObjectRequest()
    {
        Bucket = "my-bucket",
        Key = "uploads/file.txt"
    });

    Console.WriteLine($"Upload URL: {result.Url}");
    ```

    Use the presigned URL to upload via HTTP:

    ```csharp title="usePresignedUpload.cs"
    using var httpClient = new HttpClient();
    var content = new ByteArrayContent(Encoding.UTF8.GetBytes("Hello, OSS!"));
    var response = await httpClient.PutAsync(result.Url, content);

    Console.WriteLine($"Upload status: {response.StatusCode}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="presignedUploadV1.cs"
    using Aliyun.OSS;

    var request = new GeneratePresignedUriRequest("my-bucket", "uploads/file.txt", SignHttpMethod.Put)
    {
        Expiration = DateTime.UtcNow.AddHours(1)
    };

    var url = ossClient.GeneratePresignedUri(request);
    Console.WriteLine($"Upload URL: {url}");
    ```
  </Tab>
</Tabs>

## Use presigned URL with curl

After generating a presigned URL, third parties can use it directly:

```bash title="Download with curl"
curl -o report.pdf "https://my-bucket.oss-cn-hangzhou.aliyuncs.com/documents/report.pdf?x-oss-..."
```

```bash title="Upload with curl"
curl -X PUT -T /path/to/file.txt "https://my-bucket.oss-cn-hangzhou.aliyuncs.com/uploads/file.txt?x-oss-..."
```

<Note>
  Presigned URLs use V4 signatures by default with the V2 SDK. The URL expires after the specified duration (default is 1 hour). The maximum allowed expiration is 7 days.
</Note>
