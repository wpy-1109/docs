# Access control

## Bucket ACL

ACL types for buckets:

| ACL | Value | Description |
|-----|-------|-------------|
| Private | `private` | Only the owner and authorized users can read/write |
| Public Read | `public-read` | Anyone can read; only owner can write |
| Public Read-Write | `public-read-write` | Anyone can read and write |

### Set bucket ACL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="setBucketAcl.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var cfg = OSS.Configuration.LoadDefault();
    cfg.CredentialsProvider = new OSS.Credentials.EnvironmentVariableCredentialsProvider();
    cfg.Region = "cn-hangzhou";

    using var client = new OSS.Client(cfg);

    var result = await client.PutBucketAclAsync(new OSS.Models.PutBucketAclRequest()
    {
        Bucket = "my-bucket",
        Acl = "private"
    });

    Console.WriteLine($"Bucket ACL set. Status: {result.StatusCode}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="setBucketAclV1.cs"
    using Aliyun.OSS;

    ossClient.SetBucketAcl("my-bucket", CannedAccessControlList.Private);
    ```
  </Tab>
</Tabs>

### Get bucket ACL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="getBucketAcl.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var result = await client.GetBucketAclAsync(new OSS.Models.GetBucketAclRequest()
    {
        Bucket = "my-bucket"
    });

    Console.WriteLine($"ACL: {result.ACL}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="getBucketAclV1.cs"
    var acl = ossClient.GetBucketAcl("my-bucket");
    Console.WriteLine($"ACL: {acl.ACL}");
    ```
  </Tab>
</Tabs>

## Object ACL

### Set object ACL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="setObjectAcl.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var result = await client.PutObjectAclAsync(new OSS.Models.PutObjectAclRequest()
    {
        Bucket = "my-bucket",
        Key = "example.txt",
        Acl = "public-read"
    });

    Console.WriteLine($"Object ACL set. Status: {result.StatusCode}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="setObjectAclV1.cs"
    ossClient.SetObjectAcl("my-bucket", "example.txt", CannedAccessControlList.PublicRead);
    ```
  </Tab>
</Tabs>

### Get object ACL

<Tabs>
  <Tab title="V2 SDK (Recommended)">
    ```csharp title="getObjectAcl.cs"
    using OSS = AlibabaCloud.OSS.V2;

    var result = await client.GetObjectAclAsync(new OSS.Models.GetObjectAclRequest()
    {
        Bucket = "my-bucket",
        Key = "example.txt"
    });

    Console.WriteLine($"Object ACL: {result.ACL}");
    ```
  </Tab>
  <Tab title="V1 SDK">
    ```csharp title="getObjectAclV1.cs"
    var acl = ossClient.GetObjectAcl("my-bucket", "example.txt");
    Console.WriteLine($"Object ACL: {acl.ACL}");
    ```
  </Tab>
</Tabs>

<Warning>
  Use `public-read` and `public-read-write` ACLs with caution. They expose your data to the internet. For most use cases, keep ACL as `private` and use presigned URLs or RAM policies for access control.
</Warning>
