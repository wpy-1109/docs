# Download objects

## Generate a signed download URL

```javascript title="signedUrl.js"
const url = client.signatureUrl('documents/report.pdf', {
  expires: 3600 // URL valid for 1 hour
});

// Open in a new tab
window.open(url);
```

## Force download (Content-Disposition)

```javascript title="forceDownload.js"
const url = client.signatureUrl('documents/report.pdf', {
  expires: 3600,
  response: {
    'content-disposition': 'attachment; filename="report.pdf"'
  }
});

// Trigger download
const a = document.createElement('a');
a.href = url;
a.download = 'report.pdf';
a.click();
```

## Download to memory

```javascript title="downloadToMemory.js"
const result = await client.get('example.txt');
console.log('Content:', result.content.toString());
```

## Range download

```javascript title="rangeDownload.js"
const result = await client.get('large-file.bin', {
  headers: {
    Range: 'bytes=0-1023' // First 1 KB
  }
});
console.log('Downloaded bytes:', result.content.length);
```

## Preview an image

```javascript title="previewImage.js"
// Generate a URL with image processing parameters
const url = client.signatureUrl('photos/image.jpg', {
  expires: 3600,
  process: 'image/resize,w_200' // Resize to 200px width
});

document.getElementById('preview').src = url;
```
