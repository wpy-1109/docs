# Upload objects

Upload data to OSS using simple upload (PutObject) for files up to 5 GB. For larger files, use [multipart upload](/sdks/java/multipart-upload).

## Upload a local file

<Tabs>
  <Tab title="V1 SDK">
    ```java title="UploadFile.java"
    import com.aliyun.oss.model.PutObjectRequest;
    import java.io.File;

    PutObjectRequest request = new PutObjectRequest(
        bucketName, "documents/report.pdf", new File("/path/to/report.pdf"));
    ossClient.putObject(request);
    ```
  </Tab>
  <Tab title="V2 SDK">
    ```java title="UploadFileV2.java"
    PutObjectResult result = client.putObject(PutObjectRequest.newBuilder()
        .bucket(bucketName)
        .key("documents/report.pdf")
        .body(BinaryData.fromFile(Paths.get("/path/to/report.pdf")))
        .build());
    System.out.printf("ETag: %s%n", result.eTag());
    ```
  </Tab>
</Tabs>

## Upload a string

```java title="UploadString.java"
String content = "Hello OSS!";
ossClient.putObject(bucketName, "example.txt",
    new ByteArrayInputStream(content.getBytes()));
```

## Upload with metadata

```java title="UploadMetadata.java"
ObjectMetadata metadata = new ObjectMetadata();
metadata.setContentType("image/jpeg");
metadata.addUserMetadata("x-oss-meta-author", "example-user");

PutObjectRequest request = new PutObjectRequest(
    bucketName, "photos/image.jpg", new File("/path/to/image.jpg"));
request.setMetadata(metadata);
ossClient.putObject(request);
```

## Append upload

Append data to an existing appendable object:

```java title="AppendUpload.java"
AppendObjectRequest appendRequest = new AppendObjectRequest(
    bucketName, "append-log.txt",
    new ByteArrayInputStream("First entry\n".getBytes()));
appendRequest.setPosition(0L);

AppendObjectResult appendResult = ossClient.appendObject(appendRequest);

// Append more data using the next position
appendRequest = new AppendObjectRequest(
    bucketName, "append-log.txt",
    new ByteArrayInputStream("Second entry\n".getBytes()));
appendRequest.setPosition(appendResult.getNextPosition());
ossClient.appendObject(appendRequest);
```

## Upload with progress callback

```java title="UploadProgress.java"
ossClient.putObject(new PutObjectRequest(bucketName, objectName, new File(filePath))
    .withProgressListener(progressEvent -> {
        long bytes = progressEvent.getBytes();
        System.out.println("Bytes transferred: " + bytes);
    }));
```

## Upload with server callback

```java title="UploadCallback.java"
PutObjectRequest request = new PutObjectRequest(bucketName, objectName, filePath);

Map<String, String> callbackParams = new HashMap<>();
callbackParams.put("callbackUrl", "https://your-server.com/callback");
callbackParams.put("callbackBodyType", "application/json");
callbackParams.put("callbackBody",
    "{\"bucket\":${bucket},\"object\":${object},\"size\":${size}}");
request.setCallbackParam(callbackParams);

ossClient.putObject(request);
```

## Upload limits

| Constraint | Value |
|------------|-------|
| Max size (PutObject) | 5 GB |
| Max size (Multipart) | 48.8 TB |
| Min part size | 100 KB |

<Note>
If an object with the same key already exists, PutObject will overwrite it. Use the `x-oss-forbid-overwrite` header to prevent accidental overwrites.
</Note>
