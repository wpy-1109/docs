# Access control

## Bucket ACL

```java title="BucketAcl.java"
// Set bucket ACL
ossClient.setBucketACL(bucketName, CannedAccessControlList.Private);

// Get bucket ACL
AccessControlList acl = ossClient.getBucketAcl(bucketName);
System.out.println("ACL: " + acl.getCannedACL());
```

| ACL | Description |
|-----|-------------|
| `Private` | Only the owner can read/write |
| `PublicRead` | Anyone can read; only owner can write |
| `PublicReadWrite` | Anyone can read and write |

## Object ACL

```java title="ObjectAcl.java"
ossClient.setObjectAcl(bucketName, objectName, CannedAccessControlList.PublicRead);

ObjectAcl acl = ossClient.getObjectAcl(bucketName, objectName);
System.out.println("ACL: " + acl.getPermission());
```

## Bucket policy

```java title="BucketPolicy.java"
String policyText = "{"
    + "\"Version\": \"1\","
    + "\"Statement\": [{"
    + "    \"Effect\": \"Allow\","
    + "    \"Principal\": [\"*\"],"
    + "    \"Action\": [\"oss:GetObject\"],"
    + "    \"Resource\": [\"acs:oss:*:*:" + bucketName + "/public/*\"]"
    + "}]"
    + "}";

ossClient.setBucketPolicy(bucketName, policyText);

// Get policy
GetBucketPolicyResult result = ossClient.getBucketPolicy(bucketName);
System.out.println("Policy: " + result.getPolicyText());

// Delete policy
ossClient.deleteBucketPolicy(bucketName);
```

## CORS configuration

```java title="CorsConfig.java"
SetBucketCORSRequest corsRequest = new SetBucketCORSRequest(bucketName);

CORSRule rule = new CORSRule();
rule.addAllowedOrigin("https://example.com");
rule.addAllowedMethod("GET");
rule.addAllowedMethod("PUT");
rule.addAllowedMethod("POST");
rule.addAllowedHeader("*");
rule.setMaxAgeSeconds(3600);

corsRequest.addCorsRule(rule);
ossClient.setBucketCORS(corsRequest);
```

## STS temporary credentials

```java title="StsAuth.java"
String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
String accessKeyId = "<temporaryAccessKeyId>";
String accessKeySecret = "<temporaryAccessKeySecret>";
String securityToken = "<securityToken>";

OSS ossClient = new OSSClientBuilder().build(
    endpoint, accessKeyId, accessKeySecret, securityToken);
```
