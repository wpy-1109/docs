# Lifecycle rules

Automatically transition objects to cheaper storage classes or delete them based on age or access patterns.

## Set lifecycle rules

### Basic expiration rule

```java title="SetLifecycle.java"
SetBucketLifecycleRequest request = new SetBucketLifecycleRequest(bucketName);

// Delete objects with prefix "logs/" after 30 days
LifecycleRule rule = new LifecycleRule("rule1", "logs/",
    LifecycleRule.RuleStatus.Enabled);
rule.setExpirationDays(30);
request.AddLifecycleRule(rule);

ossClient.setBucketLifecycle(request);
```

### Storage transition rule

```java title="TransitionRule.java"
LifecycleRule rule = new LifecycleRule("rule2", "archive/",
    LifecycleRule.RuleStatus.Enabled);

LifecycleRule.StorageTransition transition = new LifecycleRule.StorageTransition();
transition.setStorageClass(StorageClass.IA);
transition.setExpirationDays(60);

List<LifecycleRule.StorageTransition> transitions = new ArrayList<>();
transitions.add(transition);
rule.setStorageTransition(transitions);
```

### Access time-Based transition

```java title="AccessTimeTransition.java"
// Enable access monitoring first
ossClient.putBucketAccessMonitor(bucketName,
    AccessMonitor.AccessMonitorStatus.Enabled.toString());

LifecycleRule.StorageTransition transition = new LifecycleRule.StorageTransition();
transition.setStorageClass(StorageClass.IA);
transition.setExpirationDays(30);
transition.setIsAccessTime(true);           // Transition based on last access time
transition.setReturnToStdWhenVisit(false);  // Don't convert back when accessed
transition.setAllowSmallFile(true);         // Apply to small files too
```

## Get lifecycle rules

```java title="GetLifecycle.java"
List<LifecycleRule> rules = ossClient.getBucketLifecycle(bucketName);
for (LifecycleRule rule : rules) {
    System.out.printf("Rule: %s, Prefix: %s, Status: %s, Expiration: %d days%n",
        rule.getId(), rule.getPrefix(), rule.getStatus(), rule.getExpirationDays());
}
```

## Delete lifecycle rules

```java title="DeleteLifecycle.java"
ossClient.deleteBucketLifecycle(bucketName);
```

## Lifecycle actions

| Action | Description |
|--------|-------------|
| **Transition** | Move objects to IA, Archive, Cold Archive, or Deep Cold Archive |
| **Expiration** | Delete objects after specified days or on a specific date |
| **AbortMultipartUpload** | Clean up incomplete multipart uploads |
| **NoncurrentVersionTransition** | Transition noncurrent versions (versioned buckets) |
| **NoncurrentVersionExpiration** | Delete noncurrent versions |
