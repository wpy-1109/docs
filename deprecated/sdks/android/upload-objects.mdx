# Upload objects

## Simple upload

```java title="simpleUpload.java"
PutObjectRequest put = new PutObjectRequest("my-bucket", "example.txt", "/path/to/file.txt");

put.setProgressCallback((request, currentSize, totalSize) -> {
    double progress = (double) currentSize / totalSize * 100;
    Log.d("OSS", String.format("Progress: %.1f%%", progress));
});

OSSAsyncTask task = oss.asyncPutObject(put, new OSSCompletedCallback<PutObjectRequest, PutObjectResult>() {
    @Override
    public void onSuccess(PutObjectRequest request, PutObjectResult result) {
        Log.d("OSS", "Upload complete. ETag: " + result.getETag());
    }
    @Override
    public void onFailure(PutObjectRequest request, ClientException ce, ServiceException se) {
        Log.e("OSS", "Upload failed");
    }
});
```

## Resumable upload

For large files, use resumable upload with automatic checkpoint:

```java title="resumableUpload.java"
ResumableUploadRequest request = new ResumableUploadRequest(
    "my-bucket", "videos/large-video.mp4", "/path/to/large-video.mp4"
);
request.setPartSize(256 * 1024); // 256 KB per part
request.setProgressCallback((req, currentSize, totalSize) -> {
    double progress = (double) currentSize / totalSize * 100;
    Log.d("OSS", String.format("Progress: %.1f%%", progress));
});

oss.asyncResumableUpload(request, new OSSCompletedCallback<ResumableUploadRequest, ResumableUploadResult>() {
    @Override
    public void onSuccess(ResumableUploadRequest request, ResumableUploadResult result) {
        Log.d("OSS", "Resumable upload complete.");
    }
    @Override
    public void onFailure(ResumableUploadRequest request, ClientException ce, ServiceException se) {
        Log.e("OSS", "Resumable upload failed");
    }
});
```

## Append upload

```java title="appendUpload.java"
AppendObjectRequest append = new AppendObjectRequest("my-bucket", "append.txt", "/path/to/part1.txt");
append.setPosition(0);

AppendObjectResult result = oss.appendObject(append);

// Append more data
AppendObjectRequest append2 = new AppendObjectRequest("my-bucket", "append.txt", "/path/to/part2.txt");
append2.setPosition(result.getNextPosition());
oss.appendObject(append2);
```
