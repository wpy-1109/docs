# Custom domain names

## Map a custom domain

```javascript title="putCname.js"
await client.putBucketCname('my-bucket', 'cdn.example.com');
```

## List custom domains

```javascript title="listCname.js"
const result = await client.getBucketCname('my-bucket');
result.forEach(cname => {
  console.log('Domain:', cname.domain, 'Status:', cname.status);
});
```

## Delete a custom domain

```javascript title="deleteCname.js"
await client.deleteBucketCname('my-bucket', 'cdn.example.com');
```

## Use a custom domain in the client

```javascript title="customDomainClient.js"
const client = new OSS({
  endpoint: 'cdn.example.com',
  accessKeyId: process.env.OSS_ACCESS_KEY_ID,
  accessKeySecret: process.env.OSS_ACCESS_KEY_SECRET,
  bucket: 'my-bucket',
  cname: true,  // Enable custom domain mode
});
```
