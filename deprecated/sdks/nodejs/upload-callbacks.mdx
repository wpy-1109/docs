# Upload callbacks

Upload callbacks allow OSS to send a notification to your application server after a successful upload.

## Simple upload with callback

```javascript title="uploadCallback.js"
const result = await client.put('example.txt', Buffer.from('content'), {
  callback: {
    url: 'https://your-server.com/oss-callback',
    body: 'bucket=${bucket}&object=${object}&size=${size}&mimeType=${mimeType}',
    contentType: 'application/x-www-form-urlencoded',
  },
});
console.log('Callback response:', result.data);
```

## Multipart upload with callback

```javascript title="multipartCallback.js"
const result = await client.multipartUpload('large-file.zip', '/path/to/file', {
  callback: {
    url: 'https://your-server.com/oss-callback',
    body: JSON.stringify({
      bucket: '${bucket}',
      object: '${object}',
      size: '${size}',
    }),
    contentType: 'application/json',
  },
});
```

## Callback parameters

| Variable | Description |
|----------|-------------|
| `${bucket}` | Bucket name |
| `${object}` | Object key |
| `${size}` | Object size in bytes |
| `${mimeType}` | Content-Type of the object |
| `${etag}` | Object ETag |
