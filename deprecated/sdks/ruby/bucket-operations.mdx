# Bucket operations

## Create a bucket

```ruby title="createBucket.rb"
require 'aliyun/oss'

client = Aliyun::OSS::Client.new(
  endpoint: 'https://oss-cn-hangzhou.aliyuncs.com',
  access_key_id: ENV['OSS_ACCESS_KEY_ID'],
  access_key_secret: ENV['OSS_ACCESS_KEY_SECRET']
)

client.create_bucket('my-bucket')
```

## List buckets

```ruby title="listBuckets.rb"
# List all buckets
buckets = client.list_buckets
buckets.each { |b| puts b.name }
```

## List buckets with prefix

```ruby title="listBucketsPrefix.rb"
buckets = client.list_buckets(:prefix => 'dev-')
buckets.each { |b| puts b.name }
```

## Check if a bucket exists

```ruby title="bucketExists.rb"
exists = client.bucket_exists?('my-bucket')
puts exists  # true or false
```

## Delete a bucket

<Warning>
  The bucket must be empty before deletion. Deleted buckets cannot be recovered, and the name may be claimed by another user.
</Warning>

```ruby title="deleteBucket.rb"
client.delete_bucket('my-bucket')
```
