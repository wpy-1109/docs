---
title: 'PutBucketAccessMonitor'
description: 'Configure access tracking for a bucket'
---

Enables or disables access tracking for a bucket, which monitors access patterns for intelligent tiering.

## Request

```
PUT /?accessmonitor HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<AccessMonitorConfiguration>
  <Status>Enabled</Status>
</AccessMonitorConfiguration>
```

## Response

**Success:** HTTP 200 with empty body.
