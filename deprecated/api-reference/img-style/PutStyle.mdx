---
title: 'PutStyle'
description: 'Create or update an image style'
---

Adds an image processing style to a bucket. Styles allow you to define reusable image processing pipelines.

## Request

```
PUT /?style&styleName={StyleName} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Style>
  <Content>image/resize,w_200,h_200/quality,q_80</Content>
</Style>
```

| Element | Type | Description |
|---------|------|-------------|
| `Content` | String | Image processing commands separated by `/`. |

## Response

**Success:** HTTP 200 with empty body.

To use a style, access the object with: `?x-oss-process=style/{StyleName}`
