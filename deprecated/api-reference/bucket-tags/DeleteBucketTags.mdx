---
title: 'DeleteBucketTags'
description: 'Delete tags from a bucket'
---


## Request

```
DELETE /?tagging HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

To delete specific tags:

```
DELETE /?tagging&keys=key1,key2 HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.
