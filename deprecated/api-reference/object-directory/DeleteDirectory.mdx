---
title: 'DeleteDirectory'
description: 'Delete a directory from a hierarchical namespace bucket'
---


## Request

```
POST /{DirectoryName}/?deleteDir HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 200 with empty body.

<Warning>
The directory must be empty before deletion.
</Warning>
