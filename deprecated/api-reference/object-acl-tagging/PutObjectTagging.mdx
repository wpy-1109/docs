---
title: 'PutObjectTagging'
description: 'Set tags for an object'
---

Sets or replaces tags for an object. An object can have up to 10 tags.

## Request

```
PUT /{ObjectName}?tagging HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Tagging>
  <TagSet>
    <Tag>
      <Key>environment</Key>
      <Value>production</Value>
    </Tag>
  </TagSet>
</Tagging>
```

## Response

**Success:** HTTP 200 with empty body.
