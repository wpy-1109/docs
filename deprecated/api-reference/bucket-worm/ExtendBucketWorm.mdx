---
title: 'ExtendBucketWorm'
description: 'Extend the retention period of a locked WORM policy'
---


## Request

```
POST /?wormExtend&wormId={WormId} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<ExtendWormConfiguration>
  <RetentionPeriodInDays>730</RetentionPeriodInDays>
</ExtendWormConfiguration>
```

## Response

**Success:** HTTP 200 with empty body.
