---
title: 'DeleteBucketPolicy'
description: 'Delete a bucket policy'
---

Deletes the policy of the specified bucket.

## Request

```
DELETE /?policy HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `AccessDenied` | 403 | You do not have permission to delete the policy. |
