---
title: 'DeleteBucket'
description: 'Delete a bucket'
---

Deletes a bucket. The bucket must be empty -- all objects, incomplete multipart uploads, and LiveChannels must be removed first.

## Request

```
DELETE / HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

**Success:** HTTP 204 with empty body.

```http
HTTP/1.1 204 No Content
x-oss-request-id: 534B371674E88A4D8906****
```

## Example

```http
DELETE / HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Date: Thu, 17 Apr 2025 03:15:40 GMT
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `BucketNotEmpty` | 409 | The bucket contains objects or incomplete multipart uploads. Delete all contents first. |
| `AccessDenied` | 403 | You do not have permission to delete this bucket. |

<Warning>
Bucket deletion is irreversible. The bucket name becomes available for other users after deletion.
</Warning>
