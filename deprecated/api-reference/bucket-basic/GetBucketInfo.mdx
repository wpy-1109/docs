---
title: 'GetBucketInfo'
description: 'Query detailed information about a bucket'
---

Returns metadata about a bucket including its location, creation date, ACL, storage class, versioning state, and encryption configuration.

## Request

```
GET /?bucketInfo HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

## Response

```xml
<?xml version="1.0" encoding="UTF-8"?>
<BucketInfo>
  <Bucket>
    <Name>my-bucket</Name>
    <Location>oss-cn-hangzhou</Location>
    <CreationDate>2025-01-15T08:30:00.000Z</CreationDate>
    <ExtranetEndpoint>oss-cn-hangzhou.aliyuncs.com</ExtranetEndpoint>
    <IntranetEndpoint>oss-cn-hangzhou-internal.aliyuncs.com</IntranetEndpoint>
    <StorageClass>Standard</StorageClass>
    <DataRedundancyType>LRS</DataRedundancyType>
    <Versioning>Enabled</Versioning>
    <TransferAcceleration>Disabled</TransferAcceleration>
    <CrossRegionReplication>Disabled</CrossRegionReplication>
    <AccessControlList>
      <Grant>private</Grant>
    </AccessControlList>
    <Owner>
      <ID>51264</ID>
      <DisplayName>51264</DisplayName>
    </Owner>
    <ServerSideEncryptionRule>
      <SSEAlgorithm>AES256</SSEAlgorithm>
    </ServerSideEncryptionRule>
    <ResourceGroupId>rg-aek27tc****</ResourceGroupId>
    <AccessMonitor>Disabled</AccessMonitor>
    <BlockPublicAccess>true</BlockPublicAccess>
  </Bucket>
</BucketInfo>
```

### Response elements

| Element | Type | Description |
|---------|------|-------------|
| `Name` | String | Bucket name. |
| `Location` | String | Region (e.g., `oss-cn-hangzhou`). |
| `CreationDate` | String | Creation time in ISO 8601 format. |
| `StorageClass` | String | Default storage class. |
| `DataRedundancyType` | String | `LRS` or `ZRS`. |
| `Versioning` | String | `Enabled`, `Suspended`, or empty. |
| `TransferAcceleration` | String | `Enabled` or `Disabled`. |
| `AccessControlList.Grant` | String | Bucket ACL. |
| `ServerSideEncryptionRule.SSEAlgorithm` | String | Encryption algorithm (`None`, `AES256`, `KMS`, `SM4`). |
| `BlockPublicAccess` | Boolean | Whether Block Public Access is enabled. |

## Example

```http
GET /?bucketInfo HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Date: Thu, 17 Apr 2025 03:15:40 GMT
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The specified bucket does not exist. |
| `AccessDenied` | 403 | You do not have permission to access this bucket. |
