---
title: 'CreateAccessPoint'
description: 'Create a bucket access point'
---

Creates an access point for a bucket. Access points provide named network endpoints with dedicated access policies.

## Request

```
PUT /?accessPoint HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Request body

```xml
<?xml version="1.0" encoding="UTF-8"?>
<CreateAccessPointConfiguration>
  <AccessPointName>my-access-point</AccessPointName>
  <NetworkOrigin>internet</NetworkOrigin>
</CreateAccessPointConfiguration>
```

| Element | Type | Required | Description |
|---------|------|----------|-------------|
| `AccessPointName` | String | Yes | Access point name (3-50 chars, lowercase, numbers, hyphens). |
| `NetworkOrigin` | String | Yes | Network origin. Values: `internet`, `vpc`. |
| `VpcConfiguration.VpcId` | String | Conditional | VPC ID. Required when `NetworkOrigin` is `vpc`. |

## Response

**Success:** HTTP 200.

```xml
<?xml version="1.0" encoding="UTF-8"?>
<CreateAccessPointResult>
  <AccessPointArn>acs:oss:cn-hangzhou:111111:accesspoint/my-access-point</AccessPointArn>
  <Alias>my-access-point-oss-cn-hangzhou-****</Alias>
</CreateAccessPointResult>
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `TooManyAccessPoints` | 400 | Exceeded the maximum number of access points (1000 per account). |
| `AccessPointAlreadyExists` | 409 | An access point with this name already exists. |
