---
title: 'GetObject'
description: 'Download an object from a bucket'
---

Downloads an object. Supports conditional requests, range downloads, and server-side image processing.

## Request

```
GET /{ObjectName} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `Range` | String | No | Byte range to download (e.g., `bytes=0-999`). |
| `If-Modified-Since` | String | No | Return object only if modified after this date. |
| `If-Unmodified-Since` | String | No | Return object only if not modified after this date. |
| `If-Match` | String | No | Return object only if ETag matches. |
| `If-None-Match` | String | No | Return object only if ETag does not match. |
| `Accept-Encoding` | String | No | Accepted encoding (e.g., `gzip`). |
| `x-oss-process` | String | No | Image processing parameters (e.g., `image/resize,w_100`). |

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `versionId` | String | No | Version ID to download (for versioned buckets). |
| `response-content-type` | String | No | Override Content-Type in the response. |
| `response-content-disposition` | String | No | Override Content-Disposition in the response. |
| `response-cache-control` | String | No | Override Cache-Control in the response. |
| `response-expires` | String | No | Override Expires in the response. |

## Response

```http
HTTP/1.1 200 OK
Content-Type: text/plain
Content-Length: 1024
ETag: "D41D8CD98F00B204E9800998ECF8****"
Last-Modified: Thu, 17 Apr 2025 03:15:40 GMT
x-oss-request-id: 534B371674E88A4D8906****

[object data]
```

For range requests, the response is HTTP 206 with `Content-Range` header.

## Examples

### Download an object

```http
GET /myfile.txt HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

### Range download

```http
GET /large-file.bin HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Range: bytes=0-1048575
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

### Download with image processing

```http
GET /photo.jpg?x-oss-process=image/resize,w_200 HTTP/1.1
Host: my-bucket.oss-cn-hangzhou.aliyuncs.com
Authorization: OSS4-HMAC-SHA256 Credential=LTAI****/20250417/cn-hangzhou/oss/aliyun_v4_request,Signature=****
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchKey` | 404 | The object does not exist. |
| `InvalidObjectState` | 403 | Archive/Cold Archive object has not been restored. |
| `InvalidRange` | 416 | The requested range is not satisfiable. |
| `PreconditionFailed` | 412 | Conditional header check failed. |
| `NotModified` | 304 | Object not modified (conditional request). |
