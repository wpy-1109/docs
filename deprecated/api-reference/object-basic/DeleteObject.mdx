---
title: 'DeleteObject'
description: 'Delete a single object'
---

Deletes a single object from a bucket. If versioning is enabled, this creates a delete marker instead of permanently removing the object.

## Request

```
DELETE /{ObjectName} HTTP/1.1
Host: {BucketName}.oss-{region}.aliyuncs.com
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `versionId` | String | No | Specific version to delete (permanently). |

## Response

**Success:** HTTP 204 with empty body.

```http
HTTP/1.1 204 No Content
x-oss-request-id: 534B371674E88A4D8906****
x-oss-delete-marker: true
x-oss-version-id: CAEQNhiBgIC4****
```

## Error codes

| Error Code | HTTP Status | Description |
|------------|-------------|-------------|
| `NoSuchBucket` | 404 | The bucket does not exist. |
| `AccessDenied` | 403 | Insufficient permissions. |
| `FileImmutable` | 400 | Object is protected by a WORM retention policy. |
