OSS does not provide a native move or rename operation. To move or rename an object, you copy it to the new location and delete the original. For buckets with hierarchical namespace enabled, OSS provides a native Rename API.

## Method 1: copy and delete

For standard buckets, move or rename by copying and then deleting the source:

<Tabs>
  <Tab title="Python">
    ```python title="move_object.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    # Step 1: Copy to new key
    client.copy_object(oss.CopyObjectRequest(
        bucket="examplebucket",
        key="new-name.txt",
        source_bucket="examplebucket",
        source_key="old-name.txt",
    ))

    # Step 2: Delete the original
    client.delete_object(oss.DeleteObjectRequest(
        bucket="examplebucket",
        key="old-name.txt",
    ))

    print("Object renamed successfully.")
    ```
  </Tab>
  <Tab title="Java">
    ```java title="MoveObject.java"
    // Step 1: Copy
    ossClient.copyObject("examplebucket", "old-name.txt",
                         "examplebucket", "new-name.txt");

    // Step 2: Delete original
    ossClient.deleteObject("examplebucket", "old-name.txt");
    ```
  </Tab>
</Tabs>

## Method 2: native rename (Hierarchical namespace)

For buckets with hierarchical namespace enabled, use the Rename API for atomic rename operations:

```bash
ossutil api rename --bucket examplebucket --key old-name.txt --rename-to new-name.txt
```

<Note>
Native rename is only available for buckets with [hierarchical namespace](/guides/advanced/hierarchical-namespace) enabled.
</Note>

## Using ossutil CLI

```bash
# Copy and delete approach
ossutil cp oss://examplebucket/old-name.txt oss://examplebucket/new-name.txt
ossutil rm oss://examplebucket/old-name.txt

# Move between buckets
ossutil cp oss://source-bucket/file.txt oss://dest-bucket/file.txt
ossutil rm oss://source-bucket/file.txt
```
