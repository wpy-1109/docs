Range download lets you retrieve a specific byte range from an object without downloading the entire file. This is useful for resuming interrupted downloads, downloading parts of large files, or implementing video seeking.

## Prerequisites

- An Alibaba Cloud account with OSS activated
- The `oss:GetObject` permission

## Using an SDK

<Tabs>
  <Tab title="Python">
    ```python title="range_download.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    # Download bytes 0 to 999 (first 1000 bytes)
    result = client.get_object(oss.GetObjectRequest(
        bucket="examplebucket",
        key="large-file.zip",
        range_header="bytes=0-999",
    ))

    data = result.body.read()
    print(f"Downloaded {len(data)} bytes")
    ```
  </Tab>
  <Tab title="Java">
    ```java title="RangeDownload.java"
    import com.aliyun.oss.*;
    import com.aliyun.oss.common.auth.*;
    import com.aliyun.oss.common.comm.SignVersion;
    import com.aliyun.oss.model.GetObjectRequest;
    import com.aliyun.oss.model.OSSObject;

    public class RangeDownload {
        public static void main(String[] args) throws Exception {
            String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
            String region = "cn-hangzhou";
            EnvironmentVariableCredentialsProvider credentialsProvider =
                CredentialsProviderFactory.newEnvironmentVariableCredentialsProvider();

            ClientBuilderConfiguration clientConfig = new ClientBuilderConfiguration();
            clientConfig.setSignatureVersion(SignVersion.V4);
            OSS ossClient = OSSClientBuilder.create()
                .endpoint(endpoint)
                .credentialsProvider(credentialsProvider)
                .clientConfiguration(clientConfig)
                .region(region)
                .build();

            try {
                GetObjectRequest request = new GetObjectRequest(
                    "examplebucket", "large-file.zip");
                request.setRange(0, 999); // First 1000 bytes
                OSSObject object = ossClient.getObject(request);
                System.out.println("Content-Range: " +
                    object.getObjectMetadata().getContentRange());
            } finally {
                ossClient.shutdown();
            }
        }
    }
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript title="rangeDownload.js"
    const OSS = require('ali-oss');

    const client = new OSS({
      region: 'oss-cn-hangzhou',
      accessKeyId: process.env.OSS_ACCESS_KEY_ID,
      accessKeySecret: process.env.OSS_ACCESS_KEY_SECRET,
      authorizationV4: true,
      bucket: 'examplebucket',
    });

    async function rangeDownload() {
      const result = await client.get('large-file.zip', {
        headers: { Range: 'bytes=0-999' },
      });
      console.log(`Downloaded ${result.content.length} bytes`);
    }

    rangeDownload();
    ```
  </Tab>
</Tabs>

## Range syntax

| Format | Description |
|--------|-------------|
| `bytes=0-499` | First 500 bytes |
| `bytes=500-999` | Bytes 500 through 999 |
| `bytes=-500` | Last 500 bytes |
| `bytes=500-` | All bytes from position 500 to end |
