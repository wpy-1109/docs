Single-connection bandwidth throttling lets you set a speed limit on individual upload or download requests. This prevents a single request from consuming all available bandwidth and ensures fair resource allocation across clients.

## How it works

Set the `x-oss-traffic-limit` header in your request to specify the bandwidth limit in bits per second. Valid range: 819200 (100 KB/s) to 838860800 (100 MB/s).

## Using an SDK

<Tabs>
  <Tab title="Python">
    ```python title="bandwidth_throttling.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    # Upload with a 5 MB/s bandwidth limit
    with open("file.txt", "rb") as f:
        result = client.put_object(oss.PutObjectRequest(
            bucket="examplebucket",
            key="file.txt",
            body=f,
            traffic_limit=5 * 1024 * 1024 * 8,  # 5 MB/s in bits
        ))

    # Download with a 1 MB/s bandwidth limit
    result = client.get_object(oss.GetObjectRequest(
        bucket="examplebucket",
        key="file.txt",
        traffic_limit=1 * 1024 * 1024 * 8,  # 1 MB/s in bits
    ))
    ```
  </Tab>
  <Tab title="Java">
    ```java title="BandwidthThrottling.java"
    // Upload with throttling
    PutObjectRequest request = new PutObjectRequest("examplebucket", "file.txt",
        new File("file.txt"));
    request.setTrafficLimit(5 * 1024 * 1024 * 8); // 5 MB/s
    ossClient.putObject(request);

    // Download with throttling
    GetObjectRequest getRequest = new GetObjectRequest("examplebucket", "file.txt");
    getRequest.setTrafficLimit(1 * 1024 * 1024 * 8); // 1 MB/s
    ossClient.getObject(getRequest);
    ```
  </Tab>
</Tabs>
