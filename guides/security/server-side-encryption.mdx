Server-side encryption (SSE) encrypts objects at rest. OSS encrypts the data before writing it to disk and decrypts it when you download. You can use OSS-managed keys (SSE-OSS), KMS-managed keys (SSE-KMS), or customer-provided keys (SSE-C).

## Encryption methods

| Method | Key Management | Use Case |
|--------|---------------|----------|
| **SSE-OSS** | OSS manages keys automatically | Default encryption, simplest setup |
| **SSE-KMS** | Key Management Service manages keys | Compliance requirements, key rotation, auditing |
| **SSE-C** | You provide keys with each request | Full control over encryption keys |

## Configure default bucket encryption

You can set a default encryption method for all new objects in a bucket.

<Tabs>
  <Tab title="Console">
    Navigate to your bucket, choose **Bucket Settings > Server-Side Encryption**, select the encryption method, and save.
  </Tab>
  <Tab title="Python">
    ```python title="sse_config.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    client.put_bucket_encryption(oss.PutBucketEncryptionRequest(
        bucket="examplebucket",
        server_side_encryption_rule=oss.ServerSideEncryptionRule(
            apply_server_side_encryption_by_default=oss.ApplyServerSideEncryptionByDefault(
                sse_algorithm="KMS",
                kms_master_key_id="your-kms-key-id",
            )
        )
    ))
    ```
  </Tab>
</Tabs>

## Upload with SSE-KMS

```python
result = client.put_object(oss.PutObjectRequest(
    bucket="examplebucket",
    key="encrypted.txt",
    body=b"Sensitive data",
    server_side_encryption="KMS",
    server_side_encryption_key_id="your-kms-key-id",
))
```
