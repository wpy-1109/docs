Same-region replication (SRR) automatically copies objects from a source bucket to a destination bucket in the same region. Use SRR for data backup, compliance, or aggregating data from multiple buckets.

## Configuration

Same-region replication follows the same setup as cross-region replication, except the source and destination buckets are in the same region.
