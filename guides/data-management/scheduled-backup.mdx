Scheduled backup creates regular backups of your bucket data using Hybrid Backup Recovery (HBR). You can restore data to a specific point in time if needed.

## Using the console

<Steps>
  <Step title="Enable backup">
    Navigate to your bucket, choose **Data Protection > Scheduled Backup**, and click **Enable**.
  </Step>
  <Step title="Configure schedule">
    Set the backup frequency, retention period, and backup vault.
  </Step>
</Steps>
