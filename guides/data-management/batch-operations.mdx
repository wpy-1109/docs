Batch operations let you perform bulk actions on large numbers of objects, such as copying, tagging, or restoring archived objects. You provide a manifest (object list) and specify the operation to perform.

## Supported operations

| Operation | Description |
|-----------|-------------|
| Copy | Copy objects to another bucket or prefix |
| Invoke function | Call a Function Compute function for each object |
| Restore | Restore archived objects |
| Tag | Add or replace tags on objects |
| Delete | Delete objects |
