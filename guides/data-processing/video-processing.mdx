OSS integrates with media processing services to transcode, snapshot, and transform video files stored in your buckets. Use video processing to generate thumbnails, convert formats, and optimize delivery for different devices and bandwidths.

## Capabilities

| Feature | Description |
|---------|-------------|
| Video snapshots | Capture a frame at a specific timestamp to generate a thumbnail |
| Transcoding | Convert between video formats (e.g., MP4, HLS, FLV) |
| Watermarking | Overlay text or image watermarks on video output |
| Resolution adjustment | Scale video to different resolutions for adaptive streaming |

## Video snapshots

Capture a video frame by appending processing parameters to the URL:

```
https://examplebucket.oss-cn-hangzhou.aliyuncs.com/video.mp4?x-oss-process=video/snapshot,t_1000,f_jpg,w_800,h_600
```

| Parameter | Description |
|-----------|-------------|
| `t_<ms>` | Timestamp in milliseconds |
| `f_<format>` | Output format: `jpg` or `png` |
| `w_<width>` | Output width in pixels |
| `h_<height>` | Output height in pixels |
| `m_fast` | Fast mode: seek to the nearest keyframe |

## Transcoding with media processing

For advanced video processing (transcoding, composition, streaming), use the Media Processing service (MPS) with OSS as the input and output storage:

<Steps>
  <Step title="Configure an MPS pipeline">
    Create a transcoding pipeline in the [MPS console](https://mps.console.alibabacloud.com/) with your OSS bucket as the input and output.
  </Step>
  <Step title="Submit a transcoding job">
    Submit a job specifying the input object key, output format, resolution, and bitrate.
  </Step>
  <Step title="Retrieve the output">
    The transcoded file is written to your specified output bucket and path.
  </Step>
</Steps>
