AI content awareness uses machine learning models to automatically analyze, tag, and categorize objects in your OSS bucket. This enables intelligent content management, search, and discovery without manual tagging.

## Capabilities

| Feature | Description |
|---------|-------------|
| Image tagging | Automatically detect and tag objects, scenes, and concepts in images |
| Face detection | Detect faces and estimate attributes like age and gender |
| OCR | Extract text from images and scanned documents |
| Scene classification | Classify images by scene type (indoor, outdoor, landscape) |
| Object detection | Identify and locate specific objects within images |

## Enable AI content awareness

<Steps>
  <Step title="Activate the service">
    Activate the Intelligent Media Management (IMM) service in your Alibaba Cloud account and bind it to your OSS bucket.
  </Step>
  <Step title="Create a project">
    Create an IMM project and associate it with the bucket you want to analyze.
  </Step>
  <Step title="Process objects">
    Use IMM APIs to analyze objects and retrieve tags, metadata, and classification results.
  </Step>
</Steps>

## Integration with OSS

| Integration Point | Description |
|-------------------|-------------|
| Data indexing | Build searchable indexes from automatically generated tags |
| Lifecycle rules | Use tags to drive lifecycle policies (e.g., archive unimportant images) |
| Event notifications | Trigger analysis on new uploads using event notifications |

<Note>
AI content awareness requires the Intelligent Media Management (IMM) service, which is billed separately from OSS. See the [IMM pricing page](https://www.alibabacloud.com/product/imm) for details.
</Note>
