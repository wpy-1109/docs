OSS image processing (IMG) lets you transform images on-the-fly by appending processing parameters to the object URL. Resize, crop, rotate, watermark, and convert images without pre-processing or storing multiple variants.

## How it works

Append the `x-oss-process` parameter to an image URL to apply transformations:

```
https://examplebucket.oss-cn-hangzhou.aliyuncs.com/photo.jpg?x-oss-process=image/resize,w_200
```

## Common operations

| Operation | Parameter | Example |
|-----------|-----------|---------|
| Resize | `resize,w_<width>,h_<height>` | `image/resize,w_200,h_200` |
| Crop | `crop,x_<x>,y_<y>,w_<w>,h_<h>` | `image/crop,x_100,y_100,w_300,h_300` |
| Rotate | `rotate,<degrees>` | `image/rotate,90` |
| Quality | `quality,q_<1-100>` | `image/quality,q_80` |
| Format conversion | `format,<fmt>` | `image/format,webp` |
| Watermark | `watermark,text_<base64>` | `image/watermark,text_SGVsbG8=` |
| Auto-orient | `auto-orient,1` | `image/auto-orient,1` |

## Chaining operations

Chain multiple operations by separating them with `/`:

```
?x-oss-process=image/resize,w_400/quality,q_80/format,webp
```

## Using the SDK

<Tabs>
  <Tab title="Python">
    ```python
    import alibabacloud_oss_v2 as oss

    result = client.get_object(oss.GetObjectRequest(
        bucket="examplebucket",
        key="photo.jpg",
        process="image/resize,w_200",
    ))

    with open("thumb.jpg", "wb") as f:
        f.write(result.body.read())
    ```
  </Tab>
  <Tab title="Java">
    ```java
    GetObjectRequest request = new GetObjectRequest("examplebucket", "photo.jpg");
    request.setProcess("image/resize,w_200");

    OSSObject ossObject = ossClient.getObject(request);
    ```
  </Tab>
</Tabs>

## Supported formats

| Input formats | Output formats |
|--------------|---------------|
| JPEG, PNG, BMP, GIF, WebP, TIFF, AVIF | JPEG, PNG, BMP, GIF, WebP, AVIF |

## Limits

- Source image size: up to 20 MB.
- Pixel limit: up to 4096 x 4096 pixels for resize operations.
- GIF processing: up to 20 MB, frame limits may apply.
