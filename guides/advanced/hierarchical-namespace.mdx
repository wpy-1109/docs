Hierarchical namespace (HNS) provides native directory support in OSS. With HNS enabled, directories are real entities that can be renamed atomically, improving performance for data lake and analytics workloads.

## Default vs hierarchical namespace

| Feature | Flat Namespace (Default) | Hierarchical Namespace |
|---------|------------------------|----------------------|
| Directories | Simulated using `/` in object keys | Real directory entries |
| Rename directory | Copy all objects + delete originals | Atomic rename operation |
| Delete directory | Delete all objects individually | Recursive delete |
| List directory | Prefix-based list with delimiter | Native directory listing |
| POSIX compatibility | Limited | Improved |

## When to use

- Data lake workloads with Spark, Hive, or Presto that require fast directory renames.
- HDFS replacement scenarios where atomic directory operations are critical.
- Applications that perform frequent directory-level operations.

<Warning>
Hierarchical namespace must be enabled at bucket creation time and cannot be changed afterward. Evaluate your requirements before creating the bucket.
</Warning>

## Create a bucket with HNS

<Steps>
  <Step title="Create the bucket">
    In the OSS console, click **Create Bucket**. Under **Advanced Settings**, enable **Hierarchical Namespace**.
  </Step>
  <Step title="Use directory operations">
    After creation, use native directory operations (create, rename, delete directories) through the console, SDKs, or APIs.
  </Step>
</Steps>

## Directory operations

| Operation | Description |
|-----------|-------------|
| Create directory | Create a real directory entry |
| Rename directory | Atomically rename a directory and all its contents |
| Delete directory | Recursively delete a directory and all its contents |
| List directory | List immediate children of a directory |

## Limitations

- HNS is irreversible once enabled.
- Some features may not be available with HNS buckets. Check compatibility before enabling.
- Available in selected regions only.
