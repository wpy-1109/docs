Cross-Origin Resource Sharing (CORS) allows web applications hosted on one domain to access OSS resources on a different domain. You must configure CORS rules on your bucket to allow browser-based access from JavaScript applications.

## When you need CORS

CORS is required when your web application makes direct requests to OSS from the browser, such as:
- Uploading files from a web page using JavaScript
- Displaying images hosted on OSS in a web app on a different domain
- Downloading files via AJAX requests

## CORS rule fields

| Field | Description |
|-------|-------------|
| **AllowedOrigin** | Origins allowed to make cross-origin requests (e.g., `https://example.com`, `*`). |
| **AllowedMethod** | HTTP methods allowed (GET, PUT, POST, DELETE, HEAD). |
| **AllowedHeader** | Headers allowed in preflight requests. |
| **ExposeHeader** | Response headers visible to the browser (e.g., `x-oss-request-id`, `ETag`). |
| **MaxAgeSeconds** | How long preflight results are cached (in seconds). |

## Using the console

<Steps>
  <Step title="Navigate to CORS Settings">
    Log on to the [OSS console](https://oss.console.alibabacloud.com/), click the bucket, then choose **Permission Control > CORS**.
  </Step>
  <Step title="Add a CORS rule">
    Click **Create Rule** and configure the allowed origins, methods, and headers.
  </Step>
</Steps>

## Using an SDK

<Tabs>
  <Tab title="Python">
    ```python title="cors_config.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    client.put_bucket_cors(oss.PutBucketCorsRequest(
        bucket="examplebucket",
        cors_configuration=oss.CORSConfiguration(
            cors_rules=[
                oss.CORSRule(
                    allowed_origins=["https://example.com"],
                    allowed_methods=["GET", "PUT", "POST"],
                    allowed_headers=["*"],
                    expose_headers=["x-oss-request-id", "ETag"],
                    max_age_seconds=3600,
                )
            ]
        )
    ))
    ```
  </Tab>
  <Tab title="Java">
    ```java title="CorsConfig.java"
    SetBucketCORSRequest request = new SetBucketCORSRequest("examplebucket");
    SetBucketCORSRequest.CORSRule rule = new SetBucketCORSRequest.CORSRule();
    rule.addAllowdOrigin("https://example.com");
    rule.addAllowedMethod("GET");
    rule.addAllowedMethod("PUT");
    rule.addAllowedHeader("*");
    rule.addExposeHeader("x-oss-request-id");
    rule.setMaxAgeSeconds(3600);
    request.addCorsRule(rule);
    ossClient.setBucketCORS(request);
    ```
  </Tab>
</Tabs>
