You can programmatically list all or a subset of your buckets to support automated tasks such as asset inventories, bulk operations, and permission audits. Buckets are returned in alphabetical order.

## Prerequisites

- SDK installed and configured ([quickstart](/quickstart))
- An Alibaba Cloud account with OSS activated
- An AccessKey pair or STS temporary credentials
- The `oss:ListBuckets` permission

## Request parameters

| Parameter | Description |
|-----------|-------------|
| `prefix` | Return only buckets whose names begin with this string. |
| `marker` | Start the list after this bucket name (used for pagination). |
| `max-keys` | Maximum number of buckets per response. Valid values: 1--1000. Default: 100. |

## Pagination

When the number of buckets exceeds `max-keys`, the response includes:
- **isTruncated**: `true` if more pages are available.
- **nextMarker**: Use this value as the `marker` for the next request.

Some SDKs (Python v2, Go v2, PHP v2, C# v2) provide a built-in **Paginator** that handles pagination automatically.

## Using the console

<Steps>
  <Step title="Open the Buckets page">
    Log on to the [OSS console](https://oss.console.alibabacloud.com/) and click **Buckets** in the left navigation pane.
  </Step>
  <Step title="View buckets">
    All buckets under your account are displayed. Click the export CSV icon to download bucket details.
  </Step>
</Steps>

## Using ossutil CLI

```bash
# List all buckets
ossutil api list-buckets

# List buckets with a specific prefix
ossutil api list-buckets --prefix example

# List buckets starting after a marker
ossutil api list-buckets --marker examplebucket

# Limit results per page
ossutil api list-buckets --max-keys 50
```

## Using an SDK

<Tabs>
  <Tab title="Python">
    ```python title="list_buckets.py"
    import alibabacloud_oss_v2 as oss

    credentials_provider = oss.credentials.EnvironmentVariableCredentialsProvider()
    cfg = oss.config.load_default()
    cfg.credentials_provider = credentials_provider
    cfg.region = "cn-hangzhou"

    client = oss.Client(cfg)

    # Use the built-in paginator for automatic pagination
    paginator = client.list_buckets_paginator()

    for page in paginator.iter_page(oss.ListBucketsRequest()):
        for bucket in page.buckets:
            print(f"Bucket: {bucket.name}, Region: {bucket.location}")
    ```

  <Tab title="Java">
    ```java title="ListBuckets.java"
    import com.aliyun.oss.*;
    import com.aliyun.oss.common.auth.*;
    import com.aliyun.oss.common.comm.SignVersion;
    import com.aliyun.oss.model.Bucket;
    import java.util.List;

    public class ListBuckets {
        public static void main(String[] args) throws Exception {
            String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
            String region = "cn-hangzhou";
            EnvironmentVariableCredentialsProvider credentialsProvider =
                CredentialsProviderFactory.newEnvironmentVariableCredentialsProvider();

            ClientBuilderConfiguration clientConfig = new ClientBuilderConfiguration();
            clientConfig.setSignatureVersion(SignVersion.V4);
            OSS ossClient = OSSClientBuilder.create()
                .endpoint(endpoint)
                .credentialsProvider(credentialsProvider)
                .clientConfiguration(clientConfig)
                .region(region)
                .build();

            try {
                List<Bucket> buckets = ossClient.listBuckets();
                for (Bucket bucket : buckets) {
                    System.out.println("Bucket: " + bucket.getName());
                }
            } finally {
                ossClient.shutdown();
            }
        }
    }
    ```
  </Tab>
  <Tab title="Go">
    ```go title="list_buckets.go"
    package main

    import (
        "context"
        "log"
        "github.com/aliyun/alibabacloud-oss-go-sdk-v2/oss"
        "github.com/aliyun/alibabacloud-oss-go-sdk-v2/oss/credentials"
    )

    func main() {
        cfg := oss.LoadDefaultConfig().
            WithCredentialsProvider(credentials.NewEnvironmentVariableCredentialsProvider()).
            WithRegion("cn-hangzhou")

        client := oss.NewClient(cfg)

        p := client.NewListBucketsPaginator(&oss.ListBucketsRequest{})

        for p.HasNext() {
            page, err := p.NextPage(context.TODO())
            if err != nil {
                log.Fatalf("Failed to get page: %v", err)
            }
            for _, b := range page.Buckets {
                log.Printf("Bucket: %v, Location: %v\n",
                    oss.ToString(b.Name), oss.ToString(b.Location))
            }
        }
    }
    ```

    const OSS = require('ali-oss');

    const client = new OSS({

      try {
        const result = await client.listBuckets();
        result.buckets.forEach(bucket => {

    }

## Limitations

You cannot use a Transfer Acceleration endpoint to list buckets. Transfer Acceleration resolves only third-level domains that include a bucket name (e.g., `https://BucketName.oss-accelerate.aliyuncs.com`), while list-buckets uses a root endpoint without a bucket name.
