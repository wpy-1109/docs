Transfer acceleration uses globally distributed access points and Alibaba Cloud's internal backbone network to optimize long-distance data transfers. Instead of traversing the public internet, requests enter the nearest Alibaba Cloud access point and travel over a high-speed internal network to the bucket's region.

## When to use

- Cross-region or cross-border uploads and downloads with high latency.
- Unstable public internet connections affecting transfer reliability.
- Large file transfers requiring consistent throughput.

## Enable transfer acceleration

<Steps>
  <Step title="Enable the feature">
    Navigate to **Bucket Settings > Transfer Acceleration** and toggle it on. It takes about 30 minutes to take effect globally.
  </Step>
  <Step title="Use the acceleration endpoint">
    Replace your standard endpoint with `oss-accelerate.aliyuncs.com` in SDK configurations, CLI commands, or browser URLs.
  </Step>
</Steps>

<Info>
After enabling transfer acceleration, the original public and internal endpoints remain fully functional. You can choose the optimal endpoint based on the user's location.
</Info>

## Acceleration endpoints

| Endpoint | Use Case |
|----------|----------|
| `oss-accelerate.aliyuncs.com` | Global acceleration (all regions) |
| `oss-accelerate-overseas.aliyuncs.com` | Regions outside Chinese mainland only |

<Warning>
Set the endpoint to `oss-accelerate.aliyuncs.com` only. Do not include the bucket name in the endpoint (e.g., do not use `mybucket.oss-accelerate.aliyuncs.com`), or domain resolution fails.
</Warning>

## Limitations

| Item | Description |
|------|-------------|
| Protocol | HTTP and HTTPS only. Non-HTTP protocols are not supported. |
| Effective time | About 30 minutes after enabling or disabling |
| Access mode | Bucket-level operations (e.g., ListBuckets) are not supported. Use the public endpoint for management operations. |

## Billing

Enabling transfer acceleration is free. You are billed for accelerated upload and download traffic when you access OSS through the acceleration endpoint. Standard public internet outbound traffic fees also apply.
